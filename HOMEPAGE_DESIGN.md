# 首页现代化设计说明

## 设计概述

根据仓库指令和现代化设计趋势，Rune Docs 首页采用简洁专注的设计理念。新设计遵循 ModelScope Docs 的视觉风格，采用卡片化布局、渐变效果和流畅动画，打造了一个专业且引人注目的文档首页。

## 页面结构

首页采用精简设计，仅包含核心区块：

1. **Hero 区块** - 首屏展示，包含标题、副标题、CTA 按钮和三个产品卡片

## 设计亮点

### Hero 区块 (`hero.tsx`)

**视觉特性：**
- 渐变背景：使用主题色的渐变叠加，营造现代感
- 径向渐变装饰：两个径向渐变圆形增加视觉深度
- 大标题渐变文字：标题使用主色到次色的渐变，形成视觉焦点
- 双 CTA 按钮：查看文档（主按钮）+ 贡献文档（次按钮）

**产品卡片增强：**
- 水平布局：图标和产品名称水平排列
- 彩色顶部装饰条：每个产品配有专属品牌色
- 图标背景：圆角容器 + 浅色背景突出图标
- 悬停效果：卡片上升 + 阴影加深 + 顶部彩条展开
- 产品专属色彩：
  - Rune 智算平台：#1877F2 (蓝色)
  - 魔哈仓库：#00A76F (绿色)
  - Boss 运营平台：#7635DC (紫色)

**产品卡片顺序：**
1. Rune 智算平台
2. 魔哈仓库
3. Boss 运营平台

**动画效果：**
- 标题从上淡入
- 副标题从上淡入
- CTA 按钮从下淡入
- 产品卡片依次从下淡入

## 技术实现

### 动画系统
- 使用 `framer-motion` 的 `m.div` 组件
- 采用 `MotionViewport` 包裹实现滚动触发
- `varFade` 变体提供统一的淡入动画
- 方向参数：`'inDown'`（从上）、`'inUp'`（从下）

### 响应式设计
- Grid 布局自动适配：`{ xs: '1fr', md: 'repeat(3, 1fr)' }`
- Stack 方向切换：`direction={{ xs: 'column', md: 'row' }}`
- 字体大小调整：`fontSize: { xs: '2rem', md: '3rem' }`
- 内边距调整：`py: { xs: 8, md: 12 }`

### 主题集成
- 使用 `useTheme()` 获取主题对象
- `alpha()` 函数创建半透明色
- `theme.customShadows` 提供预定义阴影
- 所有颜色来自主题配置，无硬编码

### 过渡效果
- 统一过渡时长：`transition: 'all 0.3s cubic-bezier(0.4, 0, 0.2, 1)'`
- 变换：`transform: 'translateY(-4px)'` 或 `'translateX(4px)'`
- 阴影：从 `z8` 到 `z16` 或 `z20`

## 设计原则

1. **一致性**：所有卡片使用统一的悬停效果和动画时长
2. **层次感**：通过字号、颜色、间距建立视觉层次
3. **品牌化**：三个产品使用专属品牌色贯穿全站
4. **可读性**：保持足够的对比度和留白
5. **响应式**：移动端和桌面端都有良好体验
6. **性能**：使用 CSS 渐变和变换，避免重排

## 后续优化建议

1. **内容完善**：根据需要可以恢复其他区块（能力总览、快速上手等）
2. **图片优化**：考虑为产品卡片添加插图或截图
3. **数据动态化**：产品描述和链接可接入配置文件
4. **国际化**：确保所有文案都通过 i18n 系统管理
5. **性能监控**：监测首页加载性能和动画流畅度
6. **深色模式**：确保所有视觉效果在深色模式下也表现良好

## 设计原则

本次设计遵循以下原则：

1. **简洁专注** - 聚焦核心产品，避免信息过载
2. **品牌化** - 强化产品线识别，统一视觉语言
3. **可读性** - 保持足够的对比度和留白
4. **响应式** - 多端完美适配
5. **性能** - 流畅不卡顿
