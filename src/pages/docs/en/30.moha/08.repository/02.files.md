# files Browse and version view

The files page is one of the core pages of repositorycollaboration, suitable for viewing the directory structure, previewing files content and tracking historical changes.

## Support capabilities

| ability | illustrate |
| --- | --- |
| filesTree browsing | Browse directories and sub-files |
| files content preview | Switch directory view or files view based on files type |
| Submit record | View the commit history under a specified citation |
| Difference comparison | View the differences between files between different versions |
| Download and pull | Combined with Token to do files download or Git operation |

## typical path

- files tree: `/file/:reference`
- Specify files: `/file/:reference/:file`
- Submission record: `/file/:reference/commits`
- Difference comparison: `/file/:reference/diff`

## Usage suggestions

1. First select the reference version or branch.
2. Browse the directory to find the target files.
3. Go to the commit record or diff page when you need to review history.
4. If there are multiple people collaboration, cooperate with the discussion page to record the change background.
