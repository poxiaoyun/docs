# discussion and collaboration

The discussion page is used to develop questions, feedback and collaboration around the repository content, and is the main entrance for team communication.

## Page capabilities

| ability | illustrate |
| --- | --- |
| discussion list | Display the discussion items under the current repository |
| New discussion | Create a new discussion theme |
| Reply to collaboration | Continue replying or editing in the details |
| Pull Request Entry | Some filescollaboration flows from the difference page into PR discussion |

## Interactive features

- The desktop side directly displays the "New discussion" button.
- The mobile version will collect the operations into the menu.
- The list supports paging or continuous loading.

## Applicable scenarios

- Document model or dataset usage issues.
- Review changes to repositoryfiles.
- Precipitate context during the collaboration process and reduce verbal communication loss.
