## Overview

GPU/CPU resource application and scheduling to optimize computing resource usage.

## Resource configuration

Select computing resources:

- **CPU**: 2-32 cores
- **Memory**: 4GB-128GB
- **GPU**: T4, V100, A100
- **Storage**: 10GB-1TB

## Resource application

1. Select resources when creating Notebook
2. Resource configuration can be adjusted during runtime
3. Pay as you go or use quotas

## Resource monitoring

View resource usage in real time:

-CPU usage
- Memory usage
- GPU utilization
- disk space

## Automatic shutdown

Set up automatic shutdown when idle to save resources:

- Automatically shut down if idle for more than 30 minutes
- Manually set shutdown time
- Save work progress

## Example: View GPU information

```python
import torch

# 检查 GPU 可用性
print(f"GPU 可用: {torch.cuda.is_available()}")
print(f"GPU 数量: {torch.cuda.device_count()}")
print(f"当前 GPU: {torch.cuda.get_device_name(0)}")
```
