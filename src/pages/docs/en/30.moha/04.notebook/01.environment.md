## Overview

Choose from a preconfigured development environment to get started quickly.

## Preconfigured image

Moha Warehouse provides a variety of pre-configured images:

- **PyTorch**: Contains the latest version of PyTorch and commonly used libraries
- **TensorFlow**: includes TensorFlow 2.x and Keras
- **General ML**: includes scikit-learn, pandas, numpy, etc.
- **Deep Learning**: includes PyTorch, TensorFlow and JAX

## Custom environment

Create a custom environment:

1. Select a base image
2. Add dependency packages
3. Configure environment variables
4. Save as template

## Environmental Management

- View installed packages
- Install new dependencies
- Export environment configuration
- Share environment to team
