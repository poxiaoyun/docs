## Overview

Mount storage volumes to save work progress and ensure data persistence.

## Storage type

- **Personal Storage**: Private storage space for each user
- **Team Storage**: Storage space shared by the team
- **Project Storage**: The storage space associated with the project

## Mount storage volume

Mount storage in Notebook:

1. Select the storage volume to mount
2. Specify the mounting path
3. Set read and write permissions

## Data synchronization

- Automatically save Notebook
- Back up data regularly
- Sync to cloud storage

## Data migration

Migrate data between different environments:

- Export data to storage volume
- Download to local
- Upload to other projects

## Example: Using Storage Volumes

```python
import os

# 检查存储卷挂载
storage_path = "/mnt/data"
if os.path.exists(storage_path):
    print(f"存储卷已挂载: {storage_path}")
    
    # 保存数据到存储卷
    with open(f"{storage_path}/data.txt", "w") as f:
        f.write("Hello, persistent storage!")
```

## Best Practices

- Save important data regularly
- Manage code using version control
- Clean unnecessary temporary files
- Back up critical data
