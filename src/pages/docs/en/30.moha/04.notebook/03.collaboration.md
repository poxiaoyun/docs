## Overview

Multi-person collaboration and code sharing improve team efficiency.

## Share Notebook

Share the notebook with team members:

1. Click the "Share" button
2. Select sharing permissions (read only/edit)
3. Generate sharing link

## Real-time collaboration

Multiple people editing Notebook at the same time:

- See other people's cursor positions
- Real-time synchronization of code modifications
- Collaboratively run cells

## Version control

Integrated Git version control:

- Initialize Git repository
- Submit code changes
- View history
- Branch management

## Comments and Discussions

Add comments in Notebook:

- Select the code to add comments
- @mention team members
- discuss and solve problems

## Example: Git operations

```bash
# 初始化 Git 仓库
!git init
!git config user.name "Your Name"
!git config user.email "your@email.com"

# 提交代码
!git add .
!git commit -m "Initial commit"
```
