## Overview

Model optimization and inference acceleration techniques to improve model efficiency.

## Model quantification

Reduce model size and increase inference speed:

```python
import torch

# 动态量化
quantized_model = torch.quantization.quantize_dynamic(
    model, {torch.nn.Linear}, dtype=torch.qint8
)

# 静态量化
model.qconfig = torch.quantization.get_default_qconfig('fbgemm')
torch.quantization.prepare(model, inplace=True)
torch.quantization.convert(model, inplace=True)
```

## Model pruning

Remove unimportant weights:

```python
import torch.nn.utils.prune as prune

# 结构化剪枝
prune.l1_unstructured(module, name="weight", amount=0.3)

# 全局剪枝
parameters_to_prune = [
    (module1, "weight"),
    (module2, "weight"),
]
prune.global_unstructured(
    parameters_to_prune,
    pruning_method=prune.L1Unstructured,
    amount=0.2,
)
```

## Model Distillation

Transfer large model knowledge to small models:

```python
def distillation_loss(student_logits, teacher_logits, temperature=2.0):
    soft_targets = F.softmax(teacher_logits / temperature, dim=-1)
    soft_pred = F.log_softmax(student_logits / temperature, dim=-1)
    return F.kl_div(soft_pred, soft_targets, reduction='batchmean') * (temperature ** 2)
```

## Inference optimization

### ONNX export

```python
import torch.onnx

torch.onnx.export(
    model,
    dummy_input,
    "model.onnx",
    opset_version=11,
    input_names=['input'],
    output_names=['output'],
    dynamic_axes={'input': {0: 'batch'}, 'output': {0: 'batch'}}
)
```

### TensorRT acceleration

```python
import tensorrt as trt

# 将 ONNX 转换为 TensorRT
builder = trt.Builder(TRT_LOGGER)
network = builder.create_network()
parser = trt.OnnxParser(network, TRT_LOGGER)
parser.parse_from_file("model.onnx")
```

## Batch processing optimization

Optimize batch size:

```python
# 动态批处理
def dynamic_batch(inputs, max_batch_size=32):
    for i in range(0, len(inputs), max_batch_size):
        batch = inputs[i:i+max_batch_size]
        yield model(batch)
```

## Performance monitoring

Monitor inference performance:

```python
import time

start = time.time()
output = model(input)
latency = time.time() - start
print(f"推理延迟: {latency*1000:.2f}ms")
```
