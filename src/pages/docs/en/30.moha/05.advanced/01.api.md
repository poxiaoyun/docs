## Overview

Automate the management of models and datasets through APIs.

## Certification

Get API Token:

1. Enter "Account Settings"
2. Generate API Token
3. Keep Token properly

```python
from moha import Client

client = Client(api_key="your_api_key")
```

## Model Management API

### Create model

```python
model = client.create_model(
    name="my-model",
    description="My awesome model",
    tags=["nlp", "transformer"],
    visibility="public"
)
```

### Upload model file

```python
client.upload_file(
    model_id=model.id,
    file_path="./model.pt",
    version="v1.0.0"
)
```

### Download model

```python
client.download_model(
    model_id="username/model-name",
    version="v1.0.0",
    save_path="./downloaded_model"
)
```

## Dataset Management API

### Create a dataset

```python
dataset = client.create_dataset(
    name="my-dataset",
    description="My training dataset",
    task="image-classification"
)
```

### Upload data

```python
client.upload_dataset(
    dataset_id=dataset.id,
    data_path="./data"
)
```

## Batch operations

Use the API to manage resources in batches:

```python
# 批量下载模型
models = ["model1", "model2", "model3"]
for model in models:
    client.download_model(model)

# 批量更新标签
for model in client.list_models():
    client.update_model(model.id, tags=["updated"])
```

## Rate Limit

- Free users: 100 requests/hour
- Paid users: 1000 requests/hour
- Enterprise users: unlimited
