# Mirror list and filtering

The image list page follows the Moha unified list framework, which is suitable for use from the two perspectives of asset discovery and image management.

## Page features

| Capabilities | Description |
| --- | --- |
| Visibility Toggle | Switch between public, private, or internal resources |
| Search | Locate image by name or keyword |
| Sort | Sort by downloads, likes and other indicators |
| Data card | View basic image information and enter the details page |

## Usage suggestions

- For public basic images to be reused across all platforms, it is recommended to maintain clear descriptions and labels.
- If the image only serves a specific team, use private or internal visibility.
- When you need to maintain image metadata, enter the warehouse settings page to update labels, aliases and README.
