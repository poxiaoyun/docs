## Application packaging

Package the custom application into Helm Chart format and prepare to publish it to the application market.

## Chart structure

```
my-app/
├── Chart.yaml          # 应用元信息
├── values.yaml         # 默认配置
├── values.schema.json  # 配置表单 Schema
├── README.md           # 应用文档
└── templates/          # Kubernetes 资源模板
    ├── deployment.yaml
    ├── service.yaml
    └── ingress.yaml
```

## Key file description

### Chart.yaml

```yaml
apiVersion: v2
name: my-app
version: 1.0.0
appVersion: "2.0"
description: 我的自定义应用
```

### values.yaml

Define user-configurable parameters and default values.

### values.schema.json

Define the UI rendering method and verification rules of the configuration form. The system will render the configuration form accordingly during deployment.

## Packaging command

```bash
helm package my-app/
# 输出: my-app-1.0.0.tgz
```

:::tip
It is recommended to use `helm lint my-app/` to check whether the Chart format is correct before packaging.
:::
