## Deploy application

Deploy apps from the App Market to your development environment.

## Deployment steps

1. Select the target application in the application market
2. Select the version to deploy
3. Configure deployment parameters:

| Parameters | Description |
|------|------|
| deployment name | name identifier of the instance |
| Target cluster | Which computing cluster to deploy to |
| Resource Specifications | Select CPU/GPU/Memory Configuration |
| Custom configuration | Modify application parameters as needed |

4. Click **Deploy** to start creating the instance
5. Wait for deployment to complete

## View deployment status

After deployment, you can view it in **My Instance**:

- **Running**: The application is ready and can be used normally
- **Deploying**: Creating and starting
- **Failure**: Deployment error, check the log to troubleshoot the reason

:::tip
Deployment of large applications (such as GPU inference service) may take several minutes, please be patient while the image is pulled and the service is started.
:::
