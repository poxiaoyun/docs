## Application maintenance

Manage deployed application instances and marketplace applications.

## Instance management

### View examples

View all deployed applications on the **My Instance** page, including running status, resource usage, and access addresses.

### Restart the instance

If the application is abnormal, you can click the **Restart** button to restart.

### Upgraded version

When there is a new version in the app market:

1. Enter the instance details page
2. Click **Upgrade**
3. Select the new version and confirm the configuration
4. Wait for the rolling upgrade to complete

### Uninstall the instance

Click **Uninstall** to release resources when they are no longer needed.

:::warning
Uninstalling deletes the instance and its data. If you need to retain data, please back up the contents of the persistent storage volume in advance.
:::

## Market application management

Administrators can manage the page on the app market:

- **Removed App**: Temporarily hidden from the market
- **Delete App**: Remove permanently (does not affect deployed instances)
- **Update information**: Modify application description, icon, etc.
