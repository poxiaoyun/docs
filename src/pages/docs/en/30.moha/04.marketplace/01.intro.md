## Application Market Introduction

The application market provides preconfigured AI application templates to help you quickly deploy commonly used model services and development tools.

## Enter path

Console → Moha → **App Market**

## Page functions

### Application classification

The App Market displays apps in the following categories:

| Classification | Description | Example |
|------|------|------|
| Inference service | Model inference deployment template | vLLM, TGI, Triton |
| Development tools | Model development and training tools | JupyterLab, VS Code Server |
| Data processing | Data preprocessing and labeling tools | Label Studio |
| Monitoring operation and maintenance | Service monitoring and logging tools | Grafana, Prometheus |

### Apply Card

Each application displays name, icon, introduction, version number and number of deployments.

## Operation Guide

1. Browse the application list and click on the application you are interested in
2. View application details (documentation, version history, configuration instructions)
3. Select the version and click **Deploy**, fill in the required parameters
4. Wait for the deployment to complete and access the application URL
