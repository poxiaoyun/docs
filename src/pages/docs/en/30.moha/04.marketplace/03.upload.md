## Upload application version

Upload a new version of an existing app market application.

## Operation steps

1. Enter the application details page → **Version Management** tab
2. Click **Upload new version**
3. Upload the new Helm Chart package (`.tgz` format)
4. The system automatically parses the version number and configuration Schema
5. After confirming the information, click **Save**

## Chart package requirements

- Standard Helm Chart packaging format (`.tgz`)
- `Chart.yaml` version number follows SemVer specification
- It is recommended to include `values.yaml` (default configuration)
- It is recommended to include `values.schema.json` (configuration form)

## Version release process

The uploaded version defaults to **Unpublished** status and needs to be manually published before it is visible to users:

1. Upload Chart → Unpublished
2. Test and verify → Release version
3. If there is any problem → Remove the version from the shelves
