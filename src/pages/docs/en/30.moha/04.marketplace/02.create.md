## Create application

Publish custom apps to the App Market.

## Preconditions

- Have the **System Administrator** or **Tenant Administrator** role
- Ready-to-apply Helm Chart package

## Creation steps

1. Enter the app market → click **Publish App**
2. Fill in the basic application information:

| Field | Description |
|------|------|
| Application name | Unique identifier of the application |
| Display name | Name displayed in the marketplace |
| Category | Select the category to which the application belongs |
| Icon | Upload application icon |
| Description | Application function description |
| Documentation | Instructions for Use and Configuration Guide |

3. Upload the Helm Chart package
4. Configure default deployment parameters
5. Click **Publish**

:::tip
Before publishing, it is recommended to verify the deployment process in a test environment to ensure that the chart package and default parameters are correct.
:::
