## Access token

Access tokens are used for authentication for Git operations and API calls.

## Create token

1. Click on the avatar in the upper right corner → **Personal Settings**
2. Enter the **Access Token** page
3. Click **Create Token**
4. Fill in the token name and select the permission scope
5. Click **Create** to generate the token

:::warning
The token is only displayed once when created, please copy and save it now. If lost, it needs to be deleted and re-created.
:::

##Permission scope

| Permissions | Description |
|------|------|
| Read | Clone and download repository contents |
| Write | Push code, upload files |
| Administration | Create and delete repositories |

## Use token

### Git operations

```bash
# 方式 1：在 URL 中携带令牌
git clone https://<用户名>:<令牌>@moha.your-domain/organizations/org/models/repo.git

# 方式 2：配置凭证缓存
git config --global credential.helper store
# 首次操作时输入用户名和令牌
```

### API calls

```bash
curl -H "Authorization: Bearer <令牌>" \
  https://moha.your-domain/api/moha/organizations/my-org/models
```

## Token Management

- **View Token List**: Displays all created tokens and their last use time
- **Delete Token**: Delete the token that is no longer in use and immediately revoke the authentication authority.
