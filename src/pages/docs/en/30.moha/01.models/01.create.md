# Model creation

Upload and register new models to Moharepository.

## Create steps

1. **Prepare modelfiles**: Make sure the modelfiles format meets the requirements
2. **Fill in basic information**: model name, description, tags, etc.
3. **Upload modelfiles**: Supports multiple upload methods (Web, CLI, API)
4. **Configure model parameters**: settings model input and output format

## Supported model formats

- PyTorch (.pt, .pth)
- TensorFlow (.pb, .h5)
- ONNX (.onnx)
- Safetensors (.safetensors)

## Command line upload

```bash
moha-cli model create --name my-model --path ./model.pt
```

## API upload

```python
from moha import Client

client = Client(api_key="your_api_key")
model = client.create_model(
    name="my-model",
    description="My awesome model",
    files=["./model.pt"]
)
```
