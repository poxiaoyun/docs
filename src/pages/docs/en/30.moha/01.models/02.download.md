## Model download

Download the model files from Moharepository to the local environment.

## Download method

### Web interface download

1. Enter the model details page → **files** tags page
2. Browse the modelfiles directory
3. Click the download button to the right of the target files

:::tip Web download is suitable for single files or small volume models. For large models (such as tens of GB), Git cloning is recommended.:::

### Git clone

Clone the complete model repository using Git:

```bash
# 克隆模型仓库
git clone https://moha.your-domain/organizations/ai-lab/models/qwen2-7b.git

# 如果模型包含大文件（LFS）
git lfs install
git clone https://moha.your-domain/organizations/ai-lab/models/qwen2-7b.git
```

### Download the specified version

```bash
# 克隆后切换到指定分支或标签
git checkout v1.0
```

## Authentication configuration

Downloading private models requires authentication:

1. Create **access token** in personal settings
2. Configure Git authentication using tokens

```bash
git clone https://<用户名>:<访问令牌>@moha.your-domain/organizations/ai-lab/models/my-private-model.git
```

:::warning Please keep the access token properly and do not disclose the token information in public code or documents.:::

## LFS largefiles

Moharepository uses Git LFS to manage large models files (such as weights files). Make sure `git-lfs` is installed when cloning:

```bash
# 安装 git-lfs
git lfs install

# 仅下载某些文件（选择性下载）
GIT_LFS_SKIP_SMUDGE=1 git clone <url>
git lfs pull --include="*.safetensors"
```
