## Model upload

Upload model files to Moharepository.

## Upload method

### Web interface upload

1. Enter the model details page → **files** tags page
2. Click the **UPLOADfiles** button
3. Select local files and upload

:::warning Web upload is restricted by browsers. It is recommended that a single files should not exceed **5GB**. Please use Git push method for large models.:::

### Git push

Use Git to push model files to repository:

```bash
# 克隆模型仓库
git clone https://moha.your-domain/organizations/ai-lab/models/my-model.git
cd my-model

# 添加模型文件
git lfs track "*.safetensors" "*.bin" "*.pt"
cp /path/to/model/* .

# 提交并推送
git add .
git commit -m "上传模型权重文件"
git push origin main
```

## Things to note

- Large models files (>100MB) will automatically use Git LFS storage
- It is recommended to use `.safetensors` format to store weights (safer and faster to load)
- After uploading, you can view all uploaded files on the files page
