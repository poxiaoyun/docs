## Introduction to Model Plaza

Model Square is the core page of Moharepository, which displays all public AI models on the platform to help you quickly discover and select suitable models.

## Enter path

Console → Moha → **Model Square**

## Page layout

### Search and filter

- **Keyword search**: Enter the model name, tags or description keywords to search
- **tagsFilter**: Filter by model type (NLP, CV, speech, etc.), framework (PyTorch, TensorFlow), license, etc. tags
- **Sort method**: Sort by popularity, latest updates, downloads, and ratings

### model card

Each model is presented as a card:

| information | illustrate |
|------|------|
| Model name | Organization/model name, such as `ai-lab/qwen2-7b` |
| describe | Model brief description |
| tags | Task type, framework, language, etc. |
| Downloads | Total number of historical downloads |
| Number of collections | The number of times it has been collected |
| Update time | Last submission time |

### Ranking list

Model Square provides multi-dimensional rankings, including trending models, popular downloads, latest releases, etc.

## Operation guide

### Browse models

1. Enter the model square page
2. Use the search bar or tags to filter for models of interest
3. Click on the model card to enter the details page

### Collection model

Click the ⭐ button on the model card or details page to add the model to your personal favorites for quick access later.

:::tip Collected models can be quickly found in "My Collection" on the homepage.:::
