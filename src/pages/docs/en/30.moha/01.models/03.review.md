## Overview

Submit review process and specification requirements.

## review process

1. **Submit for review**: After completing the model upload, click "Submit for review"
2. **Automatic Check**: The system automatically checks the model format and basic information
3. **Manual Review**: A review team evaluates model quality and compliance
4. **Audit Result**: The model can be publicly accessed after passing

## Audit standards

The model needs to meet the following conditions:

- Complete model card documentation
- Clear instructions and examples
- Comply with community norms and ethics
- Does not contain malicious code or infringing content

## review time

- Automated checks: done instantly
- Manual review: usually 1-3 working days

## Audit failure handling

If the review fails:

1. View review feedback
2. Modify the model or document as suggested
3. Resubmit for review
