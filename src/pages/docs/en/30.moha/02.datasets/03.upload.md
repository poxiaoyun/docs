## Data upload

Upload the data files to the data set in the Moha warehouse.

## Upload method

### Web interface upload

1. Enter the data set details page → **File** tab
2. Click the **Upload File** button
3. Select the file or drag the file to the upload area

:::warning
File size limit on the web is **5GB**. Please use Git to push very large data sets.
:::

### Git push upload

Suitable for large numbers of files or large data sets:

```bash
cd my-dataset

# 配置 LFS 追踪大文件
git lfs track "*.parquet" "*.csv.gz" "*.tar.gz"

# 添加并推送
git add .
git commit -m "上传数据集 v2"
git push origin main
```

## Supported formats

| Format | Extension | Applicable scenarios |
|------|--------|---------|
| CSV | `.csv` | Tabular data |
| Parquet | `.parquet` | Efficient columnar storage |
| JSONL | `.jsonl` | Structured text data |
| Image | `.jpg`, `.png` | Visual dataset |
| Audio | `.wav`, `.mp3` | Speech data set |
| Compressed package | `.tar.gz`, `.zip` | Batch data packaging |

## Notes

- Automatically detect whether the file is suitable for Git LFS when uploading
- Files with the same name will overwrite old versions (Git will retain history)
- After uploading, you can preview CSV, JSON and other text format files online
