## Dataset Overview

The Dataset Square displays all public data sets on the platform and supports search and categorized browsing.

## Enter path

Console → Moha → **Dataset**

## Page functions

### Dataset list

- **Search**: Search datasets by name and keywords
- **Filter**: filter by tag, data type, license
- **Sort**: Sort by update time, download volume, collection count

### Dataset Card

| Information | Description |
|------|------|
| Dataset Name | Organization/Dataset Name |
| Description | Introduction to the purpose of the data set |
| Tags | Data type, domain, language, etc. |
| Number of files | Number of files contained in the dataset |
| Size | Total size of the dataset |
| Update time | Last update time |

### collect

Click the ⭐ button to collect the data set for easy subsequent search and use.
