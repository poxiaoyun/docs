## Overview

Create and upload datasets to Moha warehouse.

## Creation steps

1. **Prepare data files**: Organize data files and directory structure
2. **Fill in metadata**: dataset name, description, label, etc.
3. **Upload data**: Supports multiple upload methods
4. **Configuration data format**: Describe the data format and usage

## Supported data formats

- CSV, JSON, Parquet
- Images (JPEG, PNG, WebP)
- Audio (WAV, MP3, FLAC)
- Video (MP4, AVI)
- Text (TXT, Markdown)

## Directory structure suggestions

```
my-dataset/
├── train/
│   ├── images/
│   └── labels.csv
├── val/
│   ├── images/
│   └── labels.csv
├── test/
│   ├── images/
│   └── labels.csv
└── README.md
```

## Command line upload

```bash
moha-cli dataset create --name my-dataset --path ./my-dataset
```
