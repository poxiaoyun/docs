## Create dataset

Create a new dataset warehouse in Moha warehouse.

## Operation steps

1. Enter Moha → Dataset page
2. Click the **Create Dataset** button
3. Fill in the data set information:

| Field | Description | Is it required |
|------|------|---------|
| Organization | Select the organization to which the data set belongs | Required |
| Dataset name | Unique identifier, used for URLs and Git operations | Required |
| Description | Description of the purpose of the data set | Optional |
| Visibility | Public or Private | Required |
| License | Data usage license type | Optional |

4. Click **Create** to complete

## Upload data

After creating a dataset, you can upload data files in the following ways:

### Web upload

On the dataset details page → File tab → click **Upload file**

### Git push

```bash
git clone https://moha.your-domain/organizations/my-org/datasets/my-dataset.git
cd my-dataset

# 添加数据文件
git lfs track "*.parquet" "*.csv" "*.jsonl"
cp /path/to/data/* .

git add .
git commit -m "添加训练数据"
git push origin main
```

:::tip
It is recommended to use Git LFS to manage large data files and choose efficient formats such as Parquet or JSONL.
:::
