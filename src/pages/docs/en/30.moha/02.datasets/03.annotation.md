## Overview

Annotation tools and collaboration processes support multi-person collaborative annotation and quality control.

## Annotation tools

Moha Warehouse provides a variety of annotation tools:

- **Image annotation**: bounding boxes, polygons, segmentation masks
- **Text annotation**: entity recognition, relationship extraction, emotion classification
- **Audio annotation**: speech transcription, speaker recognition
- **Video annotation**: target tracking, action recognition

## Collaboration process

1. **Assign tasks**: Assign labeling tasks to team members
2. **Online annotation**: Use online tools for annotation
3. **Quality Audit**: Review the annotation results
4. **Export data**: Export the labeled data set

## Quality Control

- Multi-person cross-validation
- Annotation consistency check
- Expert review mechanism

## Annotation Guide

Write clear annotation guidelines:

- Define labeling specifications
- Provide examples and edge cases
- Record frequently asked questions
