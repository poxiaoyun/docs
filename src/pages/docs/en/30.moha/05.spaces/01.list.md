# Space list and filter

The Space list page is interactively consistent with models, data sets, and images, but is more focused on displaying application resources that can be directly experienced or deployed.

## Page features

- Top supports keyword search.
-Supports switching between public and private resources.
- Support sorting.
- Supports using the unified filter panel to filter tasks, organizations, licenses and other conditions.

## Applicable scenarios

- Find demos or app templates you can try out directly.
- Browse interactive apps shared across your organization.
- Enter the repository details of a Space to continue viewing files, discussions, and deployment configurations.
