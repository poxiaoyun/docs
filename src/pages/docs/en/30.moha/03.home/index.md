#Homepage and personal workbench

Moha's homepage is not a simple landing page, but a set of workbench centered around "discovering resources, managing personal assets, and entering organizational space".

## Page composition

| Region | Description |
| --- | --- |
| Home page overview | Recommended resources, banners, quick creation portal, announcements |
| Created by me | View the models, datasets, images, and spaces you created by resource type |
| My Collection | Aggregate collected resources for easy access again |
| Access token | Copy the access token in the current login state |
| Announcement | View platform announcement details |
| Organization home page | Enter the resource list under an organization |
