#Homepage overview

The homepage overview page integrates resource discovery and quick operation capabilities, and is suitable as the first page after entering Moha.

## Enter path

Moha -> Home -> Overview

Path: `/moha/home/overview`

## Page block

According to the front-end implementation, the homepage overview mainly includes the following contents:

| Block | Description |
| --- | --- |
| Banner Carousel | Display platform banner, which can jump to external links or special content |
| Quickly create cards | Quickly create models, datasets, images and spaces |
| Recommended resources | Display recommended models or resource lists according to current paging conditions |
| Announcement list | Displays the announcements in effect and can open the details pop-up window |

## Quick creation

The homepage will provide four direct entrances:

- Create a new model
- Create a new data set
- Create a new image
- New Space

These four entrances will jump directly to the corresponding creation page, which is suitable for use when you have a clear goal.

## Applicable scenarios

- When new users enter the product for the first time, they should first understand what resources the platform is recommending.
- When maintainers need to quickly create a certain type of resource, they can directly enter the creation process from the home page.
- Follow announcements, events, special content or resource recommendations.
