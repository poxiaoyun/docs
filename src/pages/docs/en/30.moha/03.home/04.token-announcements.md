# Access tokens and announcements

The sidebar of Moha's homepage provides two high-frequency auxiliary pages: access token and announcement.

## Access token

Path: `/moha/home/token`

The access token page will directly display the Token corresponding to the current login state and provide a copy button. Common uses:

- Git pulls or pushes resource repositories.
- Call Moha interface through script or SDK.
- Configure access credentials on other platforms.

> ⚠️ NOTE: Access tokens are sensitive credentials and are recommended to be copied and used only in a trusted environment.

## Announcement

Path: `/moha/home/announcements`

The announcement page displays the currently effective announcements in a table format, supporting:

- Search by title
- View announcement summary
- Check the effective time of the announcement
- Open the details pop-up window to read the full text

Announcements are suitable for publishing platform updates, maintenance notices, event information, or governance policy adjustments.
