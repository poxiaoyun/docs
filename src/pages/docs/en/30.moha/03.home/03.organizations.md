# organize space

Organizational spaces are used to manage resources from a team perspective. Under each organization, you can view four types of resources: models, data sets, images, and Spaces, and share organization avatars, descriptions, and resource statistics.

## Enter path

Moha homepage sidebar -> Organization homepage

Path: `/moha/organizations/:organization`

## Page structure

| Block | Description |
| --- | --- |
| Organization information card | Displays organization avatar, alias and description |
| Resources tab | Switch between models, datasets, images, and spaces |
| Organization resource list | Display resources of this type under the current organization |
| List of other organizations | Easy to switch between multiple organizations |

## Resource view

The top label of the organization page will display various resource statistics, suitable for:

- View resource accumulation from a team perspective.
- Unified browsing of the organization's public or internal shared resources.
- Enter a specific repository to continue working on files, discussions and settings.

> 💡 Tip: If you only want to see the resources you created, it is more suitable to use the "I Created" or "My Resources" view; if you want to see the entire team's assets, please use the organization space.
