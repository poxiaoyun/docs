# Resources I created

"What I Created" is the personal asset view in Moha, which is used to view the models, data sets, mirrors and Spaces you maintain by resource type.

## Related entries

| Entrance | Description |
| --- | --- |
| `/moha/home/created-by-me` | View resources created by me aggregated by type |
| `/moha/my/models` | View the models I created |
| `/moha/my/datasets` | View the datasets I created |
| `/moha/my/images` | View the images I created |
| `/moha/my/spaces` | View the Spaces I created |

## Page features

-Support switching by resource type.
- Supports creating corresponding resources directly from the current page.
- The top tab will display the statistical quantities of different resource types.
- The list on the right follows the unified data card style to facilitate quick access to details.

## Applicable scenarios

- Return to the warehouse you are responsible for maintaining and continue to update content.
- Quickly check whether a certain type of resource has been created successfully.
- Unified management of different types of assets from a personal perspective.
