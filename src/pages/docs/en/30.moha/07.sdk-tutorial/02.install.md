## Install

### Basic installation

The simplest installation method:

```bash
pip install xiaoshiai-hub
```

### Install specific version

```bash
# 安装指定版本（例如 1.2.0）
pip install xiaoshiai-hub==1.2.0

# 安装最新开发版本（来自 GitHub）
pip install git+https://github.com/your-org/moha-sdk.git
```

## Dependency requirements

### Python version

- **MINIMUM REQUIREMENTS**: Python 3.7+
- **Recommended version**: Python 3.9+ (better performance and compatibility)

### Core dependencies

The SDK will automatically install the following dependencies during installation:

| Dependencies | Version range | Purpose |
|-----|--------|------|
| `requests` | >=2.20.0, <3.0.0 | HTTP request library |
| `tqdm` | >=4.62.0, <5.0.0 | Progress bar display |
| `cryptography` | >=46.0.3, <47.0.0 | Encryption/decryption operations |

### Optional dependencies

#### Upload function - Git support

If you want to use `upload_folder` or Git-based operations, you need to install:

```bash
pip install xiaoshiai-hub[upload]
```

This will additionally install:
- `gitpython>=3.1.0,<4.0.0` — Git operation library

#### Encryption Features - Enterprise Grade Encryption

AES-256-CTR encryption is built into the SDK, but enterprise-grade SM4-CTR encryption requires an external library:

```bash
# 注意：xpai-enc 不在 PyPI，需从 GitHub 安装
pip install git+https://github.com/poxiaoyun/xpai-enc.git
```

### Development dependencies (contributors)

If you want to participate in development or run tests:

```bash
pip install xiaoshiai-hub[dev]
```

## Environment variable configuration

### Basic environment variables

The SDK is configured via the following environment variables:

| Environment variables | Description | Example | Priority |
|---------|------|------|--------|
| `MOHA_ENDPOINT` | Moha Hub service address | `https://moha.your-domain` | Highest priority in CLI |
| `MOHA_TOKEN` | API Token (recommended) | `sk_xxxxxxxxxxxx` | 1 |
| `MOHA_USERNAME` | Username | `your_username` | 2 |
| `MOHA_PASSWORD` | Password | `your_password` | 2 |

### Set environment variables

#### Linux / macOS

```bash
# 临时设置（仅当前 shell 会话）
export MOHA_ENDPOINT=https://moha.your-domain
export MOHA_TOKEN=your_token

# 永久设置（添加到 ~/.bashrc 或 ~/.zshrc）
echo 'export MOHA_ENDPOINT=https://moha.your-domain' >> ~/.bashrc
source ~/.bashrc
```

#### Windows

```powershell
# 临时设置（仅当前 PowerShell 会话）
$env:MOHA_ENDPOINT = "https://moha.your-domain"
$env:MOHA_TOKEN = "your_token"

# 永久设置
[Environment]::SetEnvironmentVariable("MOHA_ENDPOINT", "https://moha.your-domain", "User")
```

### Configure in Python code

If you don't want to use environment variables, you can set them directly in the code:

```python
import os

# 设置环境变量
os.environ["MOHA_ENDPOINT"] = "https://moha.your-domain"
os.environ["MOHA_TOKEN"] = "your_token"

# 或者直接在初始化时传参
from xiaoshiai_hub import HubClient

client = HubClient(
    base_url="https://moha.your-domain",
    token="your_token"
)
```

## Verify installation

### Check SDK version

```bash
python -c "from xiaoshiai_hub import __version__; print(__version__)"
```

### Test connection

```python
from xiaoshiai_hub import HubClient

client = HubClient(base_url="https://moha.your-domain")
models = client.list_repositories("models")
print(f"✓ 连接成功，获取到 {len(models)} 个模型")
```

### Testing and Certification

```python
from xiaoshiai_hub import load_token

try:
    token, base_url, username = load_token()
    print(f"✓ 认证成功，用户: {username}")
except Exception as e:
    print(f"✗ 认证失败: {e}")
```

## Offline deployment

### Download dependencies locally

On a machine with network access:

```bash
pip download xiaoshiai-hub -d ./wheels
```

### Install in offline environment

Copy the `wheels` directory to the offline machine, then:

```bash
pip install --no-index --find-links ./wheels xiaoshiai-hub
```

## Common installation issues

### Problem 1: `pip install` times out

**Symptoms**: Network timeout during installation.

**solve**:```bash
# 使用国内镜像源
pip install -i https://pypi.tsinghua.edu.cn/simple xiaoshiai-hub

# 或增加超时时间
pip install --default-timeout=1000 xiaoshiai-hub
```

### Problem 2: `cryptography` compilation failed

**symptom**:```
error: Microsoft Visual C++ 14.0 or greater is required
```

**Solution (Windows)**:
- Download and install [Microsoft C++ Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
- or install precompiled wheel:```bash
  pip install --only-binary :all: cryptography
  ```

### Question 3: Permission Denied

**symptom**:```
ERROR: Could not install packages due to an EnvironmentError: [Errno 13]
```

**solve**:```bash
# 方式 1：使用 --user flag（推荐）
pip install --user xiaoshiai-hub

# 方式 2：使用虚拟环境（最佳实践）
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或 venv\Scripts\activate  # Windows
pip install xiaoshiai-hub
```

### Issue 4: `xpai-enc` installation failed (required only when using encryption)

**symptom**:```
ERROR: Could not find a version that satisfies the requirement xpai-enc
```

**solve**:```bash
# 使用 GitHub 安装
pip install git+https://github.com/poxiaoyun/xpai-enc.git

# 如果没有 git，下载源码手动安装
git clone https://github.com/poxiaoyun/xpai-enc.git
cd xpai-enc
pip install .
```

## Virtual environment settings (recommended)

### Use venv

```bash
# 创建虚拟环境
python -m venv moha-env

# 激活虚拟环境
source moha-env/bin/activate  # Linux/macOS
# 或
moha-env\Scripts\activate  # Windows

# 安装 SDK
pip install xiaoshiai-hub

# 使用完后退出虚拟环境
deactivate
```

### Using conda

```bash
# 创建 conda 环境
conda create -n moha-env python=3.9

# 激活环境
conda activate moha-env

# 安装 SDK
pip install xiaoshiai-hub

# 退出环境
conda deactivate
```
