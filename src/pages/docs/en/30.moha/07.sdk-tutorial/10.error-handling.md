## Exception type

Moha SDK defines the following exception classes, and the inheritance relationship is as follows:

```
HubException (基类)
├── RepositoryNotFoundError
├── FileNotFoundError
├── AuthenticationError
├── HTTPError
└── UploadError
```

### HubException

Base class for all Moha SDK exceptions.

**usage**:
```python
from xiaoshiai_hub import HubException

try:
    # SDK 操作
    pass
except HubException as e:
    print(f"Hub 操作失败: {e}")
```

### RepositoryNotFoundError

Triggered when repository does not exist or has no access rights.

**Common Scenarios**:
- repository ID misspelled
- repository has been deleted
- No access to private repository
- Hub service address configuration error

**Example**:
```python
from xiaoshiai_hub import moha_hub_download, RepositoryNotFoundError

try:
    model = moha_hub_download(
        repo_id="non-existent/model"
    )
except RepositoryNotFoundError:
    print("✗ 仓库不存在")
    # 建议：检查仓库 ID 是否正确
```

### FileNotFoundError

Triggered when files does not exist (different from Python's native FileNotFoundError).

**Common Scenarios**:
- The specified files does not exist in repository
- Branch or tags does not exist
- files has been deleted

**Example**:
```python
from xiaoshiai_hub import moha_hub_download, FileNotFoundError

try:
    config = moha_hub_download(
        repo_id="ai-lab/my-model",
        filename="non-existent.json"
    )
except FileNotFoundError:
    print("✗ 文件不存在")
    # 建议：列出仓库文件查看可用文件
```

### AuthenticationError

Triggered when authentication fails.

**Common Scenarios**:
- Token is invalid or expired
- Username/password error
- Access private repository without authentication information
- Authentication information format error

**Example**:
```python
from xiaoshiai_hub import HubClient, AuthenticationError

try:
    client = HubClient(token="invalid_token")
    client.list_repositories(repo_type="models")
except AuthenticationError:
    print("✗ 认证失败")
    print("建议：检查 Token 是否有效或已过期")
```

### HTTPError

Fired when an HTTP request fails (network error, server error, etc.).

**Common Scenarios**:
- Network connection failed
- Hub service is unavailable
- Request timeout
- Server returns 4xx/5xx errors

**Example**:
```python
from xiaoshiai_hub import HubClient, HTTPError

try:
    client = HubClient(base_url="https://invalid-domain")
    client.list_repositories(repo_type="models")
except HTTPError as e:
    print(f"✗ HTTP 错误: {e.status_code}")
    if e.status_code >= 500:
        print("服务器错误，请稍后重试")
    else:
        print("请求错误，检查参数")
```

### UploadError

Fired when the upload operation fails.

**Common Scenarios**:
- Upload files size exceeds limit
- Not enough disk space
- Network outage
- Insufficient permissions

**Example**:
```python
from xiaoshiai_hub import upload_file, UploadError

try:
    upload_file(
        path_file="huge_file.bin",
        path_in_repo="huge_file.bin",
        repo_id="my-org/repo"
    )
except UploadError as e:
    print(f"✗ 上传失败: {e}")
    # 建议：检查文件大小、网络连接
```

## Error handling best practices

### Mode 1: Basic error catching

```python
from xiaoshiai_hub import (
    moha_hub_download,
    HubException
)

try:
    model = moha_hub_download(
        repo_id="ai-lab/model",
        filename="config.json"
    )
except HubException as e:
    print(f"错误: {e}")
    # 根据错误类型处理
```

### Mode 2: Detailed error handling

```python
from xiaoshiai_hub import (
    moha_hub_download,
    RepositoryNotFoundError,
    FileNotFoundError,
    AuthenticationError,
    HTTPError
)

try:
    model = moha_hub_download(
        repo_id="ai-lab/model",
        filename="config.json"
    )
except RepositoryNotFoundError:
    print("✗ 仓库不存在或无权限")
    # 处理策略：检查仓库名、权限
except FileNotFoundError:
    print("✗ 文件不存在")
    # 处理策略：列出可用文件
except AuthenticationError:
    print("✗ 认证失败")
    # 处理策略：重新登录
except HTTPError as e:
    print(f"✗ 网络错误: {e.status_code}")
    # 处理策略：重试、切换网络
```

### Mode 3: Retry mechanism

```python
import time
from xiaoshiai_hub import moha_hub_download, HTTPError

def download_with_retry(repo_id, filename, max_retries=3):
    for attempt in range(max_retries):
        try:
            return moha_hub_download(
                repo_id=repo_id,
                filename=filename
            )
        except HTTPError as e:
            if attempt < max_retries - 1:
                wait_time = 2 ** attempt  # 指数退避
                print(f"第 {attempt + 1} 次失败，{wait_time} 秒后重试...")
                time.sleep(wait_time)
            else:
                raise

# 使用
try:
    model = download_with_retry("ai-lab/model", "config.json")
except Exception as e:
    print(f"最终失败: {e}")
```

### Mode 4: Logging

```python
import logging
from xiaoshiai_hub import HubClient, HubException

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

client = HubClient()

try:
    models = client.list_repositories(repo_type="models")
    logger.info(f"成功获取 {len(models)} 个模型")
except HubException as e:
    logger.error(f"获取模型列表失败: {e}", exc_info=True)
    raise
```

## Frequently Asked Questions (FAQ)

### Q1: How to solve "AuthenticationError: Authentication failed"?

**symptom**:
```
xiaoshiai_hub.exceptions.AuthenticationError: Authentication failed
```

**Possible reasons**:
1. Token is invalid or expired
2. Token is not settings
3. Username/password error
4. Hub address configuration error

**Solution steps**:

```bash
# 步骤 1：检查 Token 是否设置
echo $MOHA_TOKEN

# 步骤 2：重新生成 Token
moha login

# 步骤 3：验证 Hub 地址
echo $MOHA_ENDPOINT

# 步骤 4：测试认证
python -c "from xiaoshiai_hub import load_token; print(load_token())"
```

### Q2: How to solve "RepositoryNotFoundError"?

**symptom**:
```
xiaoshiai_hub.exceptions.RepositoryNotFoundError: Resource not found
```

**Possible reasons**:
1. repository ID misspelled
2. repository has been deleted
3. No access to private repository
4. repository does not exist

**Solution steps**:

```python
from xiaoshiai_hub import HubClient

client = HubClient()

# 列出所有可用模型，找到正确的仓库 ID
models = client.list_repositories(repo_type="models")
for model in models[:10]:  # 显示前 10 个
    print(f"{model.organization}/{model.name}")
```

### Q3: How to retry if the download fails?

**symptom**:
```
HTTPError: Request failed
```

**Solution**:

```python
from xiaoshiai_hub import moha_hub_download

# 通过重试机制提高成功率
model = moha_hub_download(
    repo_id="ai-lab/large-model",
    filename="model.bin"
)

# 如失败可配合重试逻辑（见下方 Q6）
```

### Q4: Failed to upload files, prompting insufficient disk space?

**symptom**:
```
UploadError: No space left on device
```

**Cause**: Insufficient server disk space

**Solution**:

```bash
# 检查服务器磁盘
df -h

# 或分割为多个上传（上传到不同仓库或目录）
moha upload org/model ./model_part1
moha upload org/model ./model_part2
```

### Q5: Failed to upload files fingerprint verification?

**symptom**:
```
UploadError: File integrity check failed
```

**Possible reasons**:
- Network interruption during upload
- Local files was modified

**Solution**:

```bash
# 重新上传（SDK 会自动续传或重新上传）
moha upload org/repo ./my_model
```

### Q6: How to deal with network instability?

**Symptoms**: Frequent `HTTPError` or `timeout`

**Solution**:

```python
import time
import random
from xiaoshiai_hub import moha_hub_download, HTTPError

def robust_download(repo_id, filename, max_retries=5):
    """网络不稳定情况下的鲁棒下载"""
    for attempt in range(max_retries):
        try:
            return moha_hub_download(
                repo_id=repo_id,
                filename=filename
            )
        except HTTPError as e:
            if attempt < max_retries - 1:
                # 指数退避 + 随机抖动
                wait_time = min(2 ** attempt + random.random(), 60)
                print(f"重试 {attempt + 1}/{max_retries}，等待 {wait_time:.1f} 秒...")
                time.sleep(wait_time)
            else:
                raise

# 使用
model = robust_download("ai-lab/model", "config.json")
```

### Q7: How to check the upload progress?

**Symptom**: Not sure if the upload will continue

**Solution**:

```bash
# 默认会显示进度条
moha upload org/repo ./my_model

# 或查看日志
moha upload org/repo ./my_model 2>&1 | tee upload.log
```

### Q8: How to clear local cache?

**Symptoms**: The local cache is too large and takes up disk space

**Solution**:

```bash
# SDK 默认 token 缓存位置
rm -rf ~/.moha/token.json

# 或直接退出登录清理本地 token
moha logout
```

### Q9: How to use proxy to access Hub?

**Symptoms**: Requires access via proxy server

**Solution**:

```python
import os
from xiaoshiai_hub import HubClient

# 设置代理
os.environ['HTTP_PROXY'] = 'http://proxy.example.com:8080'
os.environ['HTTPS_PROXY'] = 'https://proxy.example.com:8080'

client = HubClient()
# 后续请求会通过代理
```

### Q10: How to enable debug logs?

**Symptoms**: Detailed debugging information needed

**Solution**:

```python
import logging

# 启用 DEBUG 日志
logging.basicConfig(level=logging.DEBUG)

# 或仅针对 SDK
logger = logging.getLogger('xiaoshiai_hub')
logger.setLevel(logging.DEBUG)

handler = logging.StreamHandler()
handler.setLevel(logging.DEBUG)
logger.addHandler(handler)

# 现在 SDK 会输出详细日志
from xiaoshiai_hub import HubClient
client = HubClient()
client.list_repositories(repo_type="models")
```

## Performance optimization suggestions

### 1. Concurrent operations

```python
from concurrent.futures import ThreadPoolExecutor
from xiaoshiai_hub import moha_hub_download

files = ["file1.json", "file2.json", "file3.json"]

def download_file(filename):
    return moha_hub_download(
        repo_id="ai-lab/model",
        filename=filename
    )

# 并发下载多个文件
with ThreadPoolExecutor(max_workers=4) as executor:
    results = executor.map(download_file, files)
    for result in results:
        print(f"已下载: {result}")
```

### 2. Cache utilization

```python
from xiaoshiai_hub import moha_hub_download

# 第一次下载会缓存到本地
model = moha_hub_download("ai-lab/model", "config.json")

# 后续调用自动使用缓存，无需重复下载
model = moha_hub_download("ai-lab/model", "config.json")
```

### 3. Streaming large files

```python
from xiaoshiai_hub import moha_hub_download

# 下载后逐行处理，避免一次性加载到内存
filepath = moha_hub_download(
    repo_id="ai-lab/dataset",
    filename="large_data.jsonl"
)

with open(filepath) as f:
    for line in f:
        # 处理每一行
        data = json.loads(line)
```
