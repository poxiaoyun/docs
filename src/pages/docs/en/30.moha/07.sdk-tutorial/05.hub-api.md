## HubClient Overview

`HubClient` is the core class of Moha SDK and provides a way to directly call Hub API. Suitable for scenarios that require programming flexibility and fine-grained control.

### initialization

```python
from xiaoshiai_hub import HubClient

# 基础初始化（使用环境变量或本地 Token）
client = HubClient()

# 完整初始化
client = HubClient(
    base_url="https://moha.your-domain",
    token="sk_xxxxxxxxxxxx"
)

# 使用用户名/密码
client = HubClient(
    base_url="https://moha.your-domain",
    username="alice",
    password="secret"
)
```

**Parameter Description**:

| parameter | type | illustrate |
|------|------|------|
| `base_url` | `str` | Hub API address (obtained from `MOHA_ENDPOINT` environment variable by default) |
| `username` | `str` | username |
| `password` | `str` | password |
| `token` | `str` | API Token (choose one from username/password) |

## repository operation

### List repository - `list_repositories()`

List repository, support filtering by type, scope, organization.

```python
# 列出公开模型
models = client.list_repositories(repo_type="models")
for model in models:
    print(f"{model.organization}/{model.name} - {model.description}")

# 列出公开数据集
datasets = client.list_repositories(repo_type="datasets")

# 列出我创建的模型
my_models = client.list_repositories(repo_type="models", scope="create")

# 列出我收藏的模型
fav_models = client.list_repositories(repo_type="models", scope="favorite")

# 列出组织内的模型
org_models = client.list_repositories(
    repo_type="models",
    scope="organization",
    organization="my-org"
)

# 列出组织内我创建的模型
mine = client.list_repositories(
    repo_type="models",
    scope="organization",
    organization="my-org",
    mycreated=True
)
```

**parameter**:

| parameter | type | default value | illustrate |
|------|------|--------|------|
| `repo_type` | `str` | `"models"` | repository type：`"models"` / `"datasets"` / `"spaces"` |
| `scope` | `str` | `"public"` | Query range: `"public"` / `"create"` / `"favorite"` / `"organization"` |
| `organization` | `str` | `None` | Organization name (required if `scope="organization"`) |
| `mycreated` | `bool` | `False` | Show only the ones I created (valid only with `scope="organization"`) |

**Return**: `List[Repository]`

### Get repository information - `get_repository_info()`

```python
info = client.get_repository_info(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b"
)

print(f"名称: {info.name}")
print(f"组织: {info.organization}")
print(f"类型: {info.type}")
print(f"可见性: {info.visibility}")
print(f"描述: {info.description}")
```

**parameter**:

| parameter | type | illustrate |
|------|------|------|
| `organization` | `str` | Organization name |
| `repo_type` | `str` | repository Type: `"models"` / `"datasets"` / `"spaces"` |
| `repo_name` | `str` | repository name |

**Return**: `Repository`

### Create repository - `create_repository()`

```python
client.create_repository(
    organization="my-org",
    repo_type="models",
    repo_name="my-new-model",
    description="My awesome model",
    visibility="internal",
    metadata={
        "license": ["apache-2.0"],
        "tasks": ["text-generation"],
        "frameworks": ["pytorch"],
    },
    base_model=["ai-lab/qwen2-7b"],
    relationship="finetune"
)
print("✓ 仓库创建成功")
```

**parameter**:

| parameter | type | default value | illustrate |
|------|------|--------|------|
| `organization` | `str` | — | Organization name |
| `repo_type` | `str` | — | repository type |
| `repo_name` | `str` | — | repository name |
| `description` | `str` | `None` | repositoryDescription |
| `visibility` | `str` | `"internal"` | Visibility: `"public"` / `"internal"` / `"private"` |
| `metadata` | `dict` | `None` | Metadata (license, tasks, languages, tags, frameworks) |
| `base_model` | `list` | `None` | Basic model list |
| `relationship` | `str` | `None` | Relationship to the base model |

## repositoryContent Operations

### Get repository content - `get_repository_content()`

Get the files/ directory structure of repository under the specified path.

```python
# 获取仓库根目录内容
content = client.get_repository_content(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b",
    branch="main"
)

# 遍历根目录下的内容
if content.entries:
    for item in content.entries:
        item_type = "📁" if item.type == "dir" else "📄"
        print(f"{item_type} {item.path} ({item.size} bytes)")

# 获取子目录内容
sub_content = client.get_repository_content(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b",
    branch="main",
    path="weights"
)
```

**Return**: `GitContent` — Contains fields such as `name`, `path`, `type`, `size`, `entries`

### Create/Update files - `create_file()`

Applies to small text files (such as configuration files, README, etc.).

```python
client.create_file(
    organization="my-org",

    message="Add README"
```

### DOWNLOADfiles - `download_file()`

Download a single files in repository to the local.


## Version management

### Get branches and tags - `get_repository_refs()`

```python
refs = client.get_repository_refs(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b"
)

for ref in refs:
    ref_type = "分支" if ref.type == "branch" else "标签"
    default_mark = " (默认)" if ref.is_default else ""
    print(f"[{ref_type}] {ref.name}{default_mark} → {ref.hash[:8]}")
```

**Returns**: `List[Ref]` — each `Ref` contains `name`, `ref`, `type`, `hash`, `is_default` fields

### Get the default branch - `get_default_branch()`

## cryptographic operations

### Generate data key - `generate_data_key()`

Generate data keys via the KMS service for envelope encryption.

```python
from xiaoshiai_hub import HubClient

client = HubClient(token="sk_xxxx")
data_key = client.generate_data_key(
    algorithm="AES",            # "AES" 或 "SM4"
    password="your_password"
)

print(f"明文密钥长度: {len(data_key.plaintext_key)} 字节")
print(f"加密密钥: {data_key.encrypted_key[:20]}...")
```

### settingsrepository Encryption flag - `set_repository_encrypted()`

```python
client.set_repository_encrypted(
```


## Advanced usage

### Complete workflow example

```python
from xiaoshiai_hub import HubClient

# 1. 初始化客户端
client = HubClient(token="sk_xxxxxxxxxxxx")

# 2. 创建新仓库
client.create_repository(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model",
    description="Fine-tuned model",
    visibility="internal"
)
print("✓ 仓库创建成功")

# 3. 创建 README
client.create_file(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model",
    branch="main",
    file_path="README.md",
    content="# Finetuned Model\n\nFine-tuned on custom dataset.",
    message="Add README"
)
print("✓ README 已创建")

# 4. 创建开发分支
client.create_branch(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model",
    branch_name="dev",
    from_branch="main"
)
print("✓ dev 分支已创建")

# 5. 获取仓库信息
info = client.get_repository_info(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model"
)
print(f"✓ 仓库: {info.organization}/{info.name}, 可见性: {info.visibility}")
```

### Batch operation

```python
# 批量下载仓库中的多个文件
import os

files_to_download = ["config.json", "tokenizer.model", "README.md"]

for filename in files_to_download:
    try:
        local_path = os.path.join("./downloads", filename)
        os.makedirs(os.path.dirname(local_path), exist_ok=True)
        client.download_file(
            organization="ai-lab",
            repo_type="models",
            repo_name="qwen2-7b",
            branch="main",
            file_path=filename,
            local_path=local_path
        )
        print(f"✓ 已下载: {filename}")
    except Exception as e:
        print(f"✗ 下载失败: {filename} - {e}")
```

### Error handling integration

```python
from xiaoshiai_hub import (
    HubClient,
    RepositoryNotFoundError,
    AuthenticationError,
    HTTPError
)

client = HubClient()

try:
    info = client.get_repository_info(
        organization="ai-lab",
        repo_type="models",
        repo_name="non-existent"
    )
except RepositoryNotFoundError:
    print("仓库不存在")
except AuthenticationError:
    print("认证失败，请检查 Token")
except HTTPError as e:
    print(f"HTTP 错误: {e}")
```

## Data model reference

### Repository

```python
@dataclass
class Repository:
    name: str              # 仓库名称
    organization: str      # 组织名称
    owner: str             # 所有者
    creator: str           # 创建者
    type: str              # 类型: "models" / "datasets" / "spaces"
    visibility: str        # 可见性: "public" / "internal" / "private"
    description: str       # 描述
    metadata: dict         # 元数据 (license, tasks, languages 等)
    annotations: dict      # 注解
    genealogy: dict        # 模型谱系信息
    spaceMetadata: SpaceMetadata  # Space 元数据（仅 spaces）
```

### Ref

```python
@dataclass
class Ref:
    name: str         # 分支/标签名称
    ref: str          # 引用路径
    type: str         # "branch" 或 "tag"
    hash: str         # 提交哈希
    is_default: bool  # 是否为默认分支
```

### GitContent

```python
@dataclass
class GitContent:
    name: str                   # 文件/目录名
    path: str                   # 路径
    type: str                   # "file" / "dir" / "symlink" / "submodule"
    size: int                   # 文件大小（字节）
    hash: str                   # Git 哈希
    content: str                # 文件内容（仅小文件）
    entries: List[GitContent]   # 子目录内容（仅目录）
```

### DataKey

```python
@dataclass
class DataKey:
    plaintext_key: bytes   # 明文密钥
    encrypted_key: str     # 加密后的密钥（存储在文件头部）
```
