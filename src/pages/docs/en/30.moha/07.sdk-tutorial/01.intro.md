## SDK overview

Moharepository is compatible with HuggingFace Hub’s API and SDK, so you can use familiar tools to interact with the platform.

## compatibility

Moharepository is compatible with the following tools and libraries:

| tool | illustrate |
|------|------|
| `git` + `git-lfs` | Standard Git operations |
| `transformers` | Model loading and inference |
| `datasets` | Dataset loading and processing |
| `huggingface_hub` | Hub API operations |

## Configure environment

Before using the SDK, you need to configure the address of Moharepository:

```bash
# 方式 1：环境变量
export HF_ENDPOINT=https://moha.your-domain

# 方式 2：Python 代码中设置
import os
os.environ["HF_ENDPOINT"] = "https://moha.your-domain"
```

## Certification

```bash
# 配置访问令牌
export HF_TOKEN=your_access_token
```

:::tip All tools compatible with HuggingFace Hub can be used directly by configuring `HF_ENDPOINT` to point to Moharepository.:::
