## Model export

Export trained models to different formats for easy deployment and distribution.

## Export format

| Format | Applicable scenarios | illustrate |
|------|---------|------|
| Safetensors | General reasoning and storage | Safe and fast loading |
| ONNX | Cross-framework deployment | Compatible with multiple inference engines |
| GGUF | CPU inference | Suitable for tools such as llama.cpp |
| TensorRT | NVIDIA GPU inference | High performance inference optimization |

## Export as Safetensors

```python
# transformers 默认支持 safetensors
model.save_pretrained("./output", safe_serialization=True)
```

## Export to ONNX

```python
from transformers import AutoModelForSequenceClassification
from optimum.onnxruntime import ORTModelForSequenceClassification

# 导出
model = ORTModelForSequenceClassification.from_pretrained(
    "ai-lab/my-model",
    export=True
)
model.save_pretrained("./onnx-output")
```

## Upload and export models

Upload the exported model to Moharepository:

```bash
cd onnx-output
git init
git lfs track "*.onnx"
git add .
git commit -m "导出 ONNX 模型"
git remote add origin https://moha.your-domain/organizations/my-org/models/my-model-onnx.git
git push origin main
```
