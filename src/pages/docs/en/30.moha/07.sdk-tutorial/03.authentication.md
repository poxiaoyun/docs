## Certification Overview

Moha SDK supports multiple authentication methods, sorted by recommendation:

| Way | environment | Recommendation | illustrate |
|------|------|--------|------|
| **Token** | environment variables/code | ⭐⭐⭐⭐⭐ | The safest and most convenient |
| **Username/Password** | environment variables/interactive | ⭐⭐⭐⭐ | Flexible but requires saving passwords |
| **Local Storage** | `~/.moha/token.json` | ⭐⭐⭐⭐⭐ | `login` command automatic management |

## Method 1: Token authentication (recommended)

### Get Token

#### Method A: Generate in Console

1. Log in to Console → **Personal Center** → **API Key**
2. Click **Add Key**
3. Enter the Key name and select the expiration time
4. Save the displayed Token (only visible at this time)

#### Method B: Use the `login` command

```bash
moha login --username your_username --password your_password

# 成功后 Token 自动保存到 ~/.moha/token.json
```

#### Method C: Programmatic generation (advanced)

```python
from xiaoshiai_hub import login

# 使用用户名密码登录
token, token_path = login(
    username="your_username",
    password="your_password"
)
print(f"✓ Token 已保存到: {token_path}")

# 登录后，Token 自动保存
# 下次使用会自动加载
```

### Use Token for authentication

#### Method 1: Environment variables (recommended)

```bash
export MOHA_TOKEN=sk_xxxxxxxxxxxx

# 或在 .env 文件中
echo "MOHA_TOKEN=sk_xxxxxxxxxxxx" > .env
```

#### Method 2: Pass parameters in code

```python
from xiaoshiai_hub import HubClient

client = HubClient(
    base_url="https://moha.your-domain",
    token="sk_xxxxxxxxxxxx"
)

# 或者使用下载函数
from xiaoshiai_hub import moha_hub_download

model = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    token="sk_xxxxxxxxxxxx"
)
```

#### Method 3: Automatically load saved Token

```python
from xiaoshiai_hub import load_token, HubClient

# 加载之前保存的 token
token, base_url, username = load_token()

client = HubClient(token=token)
print(f"✓ 已认证为: {username}")
```

## Method 2: Username/Password Authentication

### Scenario 1: Environment variables

```bash
export MOHA_USERNAME=your_username
export MOHA_PASSWORD=your_password
```

### Scenario 2: Passing parameters in code

```python
from xiaoshiai_hub import HubClient

client = HubClient(
    base_url="https://moha.your-domain",
    username="your_username",
    password="your_password"
)
```

### Scenario 3: Interactive input (not recommended in production environment)

```python
from xiaoshiai_hub import HubClient
import getpass

username = input("用户名: ")
password = getpass.getpass("密码: ")

client = HubClient(username=username, password=password)
```

### ⚠️Safety warning

**Don't hardcode passwords in your code! ** Recommended practices:

```python
import os
from xiaoshiai_hub import HubClient

# 从环境变量读取
username = os.environ.get("MOHA_USERNAME")
password = os.environ.get("MOHA_PASSWORD")

client = HubClient(username=username, password=password)
```

## Token persistence and management

### Automatically save Token

When using the `login()` command, the Token will be automatically saved to `~/.moha/token.json`:

```python
from xiaoshiai_hub import login

token, token_path = login(
    username="your_username",
    password="your_password"
)
# Token 已保存到 ~/.moha/token.json
```

### Save Token manually

```python
from xiaoshiai_hub import save_token

# 保存 Token 到默认位置 (~/.moha/token.json)
save_token(
    token="sk_xxxxxxxxxxxx",
    username="your_username",
    base_url="https://moha.your-domain"
)
```

### Load saved token

```python
from xiaoshiai_hub import load_token

token, base_url, username = load_token()
print(f"已加载: {username} @ {base_url}")
```

### Delete saved token

```python
from xiaoshiai_hub import delete_token

delete_token()
print("Token 已删除")
```

### Custom Token storage location

The default Token is stored in `~/.moha/token.json` and can be customized:

```python
from xiaoshiai_hub import save_token, load_token

custom_token_dir = "/path/to/custom/dir"

# 保存到自定义目录
save_token(
    token="sk_xxxxxxxxxxxx",
    token_dir=custom_token_dir
)

# 从自定义目录加载
token, base_url, username = load_token(token_dir=custom_token_dir)
```

## Authentication priority

The SDK looks for authentication information in the following priority:

```
优先级顺序 (从高到低):
1. 代码中明确传参 (token / username / password)
2. 环境变量 (MOHA_TOKEN / MOHA_USERNAME / MOHA_PASSWORD)
3. 本地保存的 Token (~/.moha/token.json)
4. 无认证 → 使用公开内容
```

**Priority example**:

```python
from xiaoshiai_hub import HubClient

# 场景 1：明确传参 → 使用传参
client = HubClient(token="token_from_code")
# ✓ 使用 "token_from_code"，忽略环境变量和本地保存

# 场景 2：无明确传参，有环境变量 → 使用环境变量
# export MOHA_TOKEN=token_from_env
client = HubClient()
# ✓ 使用 "token_from_env"，忽略本地保存

# 场景 3：无明确传参，无环境变量，有本地保存 → 使用本地保存
client = HubClient()
# ✓ 自动加载 ~/.moha/token.json
```

## Common authentication scenarios

### Scenario 1: Local development

Use the `login` command to log in once, and then automatically use the saved Token:

```bash
moha login --username your_username --password your_password
# Token 已保存到 ~/.moha/token.json
```

Then load it automatically in code:

```python
from xiaoshiai_hub import HubClient

client = HubClient()  # 自动加载已保存的 Token
```

### Scenario 2: CI/CD pipeline

settings environment variable and then use it in the script:

```bash
#!/bin/bash
export MOHA_ENDPOINT=https://moha.your-domain
export MOHA_TOKEN=${MOHA_SDK_TOKEN}  # 从 CI 系统的 secrets 获取

python train.py
```

In a Python script:

```python
from xiaoshiai_hub import HubClient

client = HubClient()  # 自动读取环境变量
```

### Scenario 3: Docker container

In the Dockerfile settings environment variable:

```dockerfile
FROM python:3.9

# 安装 SDK
RUN pip install xiaoshiai-hub

# 设置认证信息（从 build args 传入）
ARG MOHA_TOKEN
ENV MOHA_TOKEN=${MOHA_TOKEN}
ENV MOHA_ENDPOINT=https://moha.your-domain

COPY train.py .
CMD ["python", "train.py"]
```

Build and run:

```bash
docker build --build-arg MOHA_TOKEN=sk_xxxxxxxxxxxx -t ml-app .
docker run ml-app
```

### Scenario 4: User accounts and permissions

Check currently authenticated user information:

```python
from xiaoshiai_hub import load_token

token, base_url, username = load_token()
print(f"当前用户: {username}")
print(f"Hub 地址: {base_url}")
```

## Debugging authentication issues

### Check certification status

```python
from xiaoshiai_hub import HubClient, load_token
import os

# 打印认证信息（不显示完整 token）
print("认证调试信息：")
print(f"1. 环境变量 MOHA_TOKEN: {bool(os.environ.get('MOHA_TOKEN'))}")
print(f"2. 环境变量 MOHA_USERNAME: {os.environ.get('MOHA_USERNAME', '未设置')}")
print(f"3. 本地保存 Token: ", end="")

try:
    token, base_url, username = load_token()
    print(f"已保存 (用户: {username})")
except:
    print("未保存")

# 测试连接
try:
    client = HubClient()
    models = client.list_repositories("models")
    print(f"✓ 认证成功")
except Exception as e:
    print(f"✗ 认证失败: {e}")
```

### Common errors and solutions

| mistake | reason | solution |
|------|------|---------|
| `AuthenticationError: Authentication failed` | Token is invalid or expired | Regenerate the Token and check whether the Token validity period has expired |
| `RepositoryNotFoundError` | repository does not exist or has no access rights | Check repository ID to confirm access rights |
| `FileNotFoundError` | files does not exist | Check whether the files path is correct |
