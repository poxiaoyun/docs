## Quick start

This guide will take you from installation to basic operation in under 5 minutes.

## 30 seconds to install

```bash
# 安装官方 SDK
pip install xiaoshiai-hub

# 验证安装
python -c "from xiaoshiai_hub import HubClient; print('✓ 安装成功')"
```

## 1 minute to configure the environment

```bash
# 设置 Moha Hub 地址（替换为你的实际域名）
export MOHA_ENDPOINT=https://moha.your-domain

# 设置认证令牌或用户名/密码
export MOHA_TOKEN=your_access_token
# 或者
export MOHA_USERNAME=your_username
export MOHA_PASSWORD=your_password
```

## Experience three major operations in 2 minutes

### Operation 1: Log in and save Token

```python
from xiaoshiai_hub import login, load_token

# 使用用户名密码登录（会自动保存 Token）
token, token_path = login(
    username="your_username",
    password="your_password"
)
print(f"✓ Token 已保存到: {token_path}")

# 验证：加载已保存的 token
token, base_url, username = load_token()
print(f"✓ 已登录用户: {username}")
```

### Operation 2: Download the model

```python
from xiaoshiai_hub import moha_hub_download

# 下载任意模型文件到本地
model_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json"
)
print(f"✓ 已下载: {model_path}")
```

### Operation 3: Upload file

```python
from xiaoshiai_hub import upload_file

# 创建测试文件
with open("my_model.txt", "w") as f:
    f.write("test model content")

# 上传到仓库
upload_file(
    path_file="my_model.txt",
    path_in_repo="my_model.txt",
    repo_id="my-org/my-model"
)
print("✓ 已上传")
```

## Quick check on common scenarios

### Scenario 1: Upload an entire folder using CLI

```bash
# 上传本地文件夹到模型仓库
moha upload ./my_model_dir my-org/my-model \
  --username your_username \
  --password your_password
```

### Scenario 2: Download the entire dataset

```python
# 下载完整数据集（包含所有文件）
from xiaoshiai_hub import snapshot_download

dataset_path = snapshot_download(
    repo_id="ai-lab/my-dataset",
    repo_type="datasets"
)
print(f"✓ 数据集已下载到: {dataset_path}")
```

### Scenario 3: Loading the model using Transformers

```python
import os
os.environ["HF_ENDPOINT"] = "https://moha.your-domain"

from transformers import AutoModelForCausalLM, AutoTokenizer

# 无缝使用，就像 HuggingFace Hub 一样
model = AutoModelForCausalLM.from_pretrained("ai-lab/qwen2-7b")
tokenizer = AutoTokenizer.from_pretrained("ai-lab/qwen2-7b")
print("✓ 模型已加载")
```

## FAQ

**Q: How to configure Moha address? **
A: Pass the environment variable `MOHA_ENDPOINT` or set it in code. See [Installation and Environment](./02.install) for details.

**Q: How to generate Token? **
A: Go to Console → Personal Center → API Key, or use the `moha login` command. For details, see [Authentication Management](./03.authentication).

**Q: How to upload large files? **
A: The SDK automatically handles large file uploads in chunks. See [Upload/Download/Sync](./06.transfer) for details.
