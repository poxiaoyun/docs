## Download Overview

Moha SDK provides two download methods:

| function | use | scene |
|------|------|------|
| `moha_hub_download()` | Download a single files | Quickly load a single modelfiles |
| `snapshot_download()` | Download the entire repository snapshot | Complete download of a dataset or model |

## Single files download - moha_hub_download()

### Basic usage

```python
from xiaoshiai_hub import moha_hub_download

# 下载单个文件
config_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json"
)

print(f"文件位置: {config_path}")
```

### Detailed explanation of parameters

```python
moha_hub_download(
    repo_id="ai-lab/qwen2-7b",    # 仓库 ID (org/name)
    filename="config.json",         # 文件名

    # 可选参数
    repo_type="models",             # "models" / "datasets" / "spaces"
    revision="main",                # 分支/标签名
    local_dir="./models",           # 下载到本地文件夹
    base_url=None,                  # 自定义 Hub 地址
    token=None,                     # API Token
    show_progress=True,             # 显示进度条
)
```

**Parameter Description**:

| parameter | type | default value | illustrate |
|------|------|--------|------|
| `repo_id` | `str` | — | repository ID in the format `org/name` |
| `filename` | `str` | — | files name to be downloaded |
| `repo_type` | `str` | `"models"` | repository type |
| `revision` | `str` | `None` | Branch name or tags name |
| `local_dir` | `str` | `None` | local save directory |
| `show_progress` | `bool` | `True` | Whether to display a progress bar |

### Specify local save location

```python
# 保存到特定目录
config_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    local_dir="./my_models"
)
# 保存到 ./my_models/config.json

# 下载模型权重文件
weights_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="model.safetensors",
    local_dir="./models"
)
# 保存到 ./models/model.safetensors
```

### Download specified version

```python
# 下载特定分支
config = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    revision="dev"
)

# 下载特定标签
v1 = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    revision="v1.0"
)
```

### Download silently (no progress bar shown)

```python
config_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    show_progress=False
)
```

## Complete repository download - snapshot_download()

### Basic usage

```python
from xiaoshiai_hub import snapshot_download

# 下载整个模型仓库
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b"
)

print(f"仓库位置: {repo_path}")
# 目录结构
# {repo_path}/
# ├── config.json
# ├── model.safetensors
# ├── tokenizer.model
# └── ...
```

### Detailed explanation of parameters

```python
snapshot_download(
    repo_id="ai-lab/qwen2-7b",     # 仓库 ID (org/name)

    # 可选参数
    repo_type="models",              # "models" / "datasets" / "spaces"
    revision=None,                   # 分支/标签名
    local_dir=None,                  # 本地保存目录
    allow_patterns=None,             # 只下载匹配的文件
    ignore_patterns=None,            # 忽略匹配的文件
    base_url=None,                   # 自定义 Hub 地址
    token=None,                      # API Token
    show_progress=True,              # 显示进度条
)
```

### Selective downloading (pattern filtering)

#### allow_patterns - only download matching files

```python
# 只下载 safetensors 权重文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns="*.safetensors"
)

# 下载多种文件格式
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns=["*.safetensors", "*.json", "*.model"]
)

# 下载特定目录
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns="weights/*"
)
```

#### ignore_patterns - ignore matching files

```python
# 下载所有文件，但忽略大文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns="*.bin"
)

# 忽略多种文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns=["*.pyc", "__pycache__", "*.tmp"]
)

# 忽略特定目录
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns=["build/*", "test/*"]
)
```

#### Use in combination

```python
# 只下载权重和配置，忽略临时文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns=["*.safetensors", "*.json"],
    ignore_patterns=["*.tmp"]
)
```

## Upload overview

Moha SDK provides two upload methods:

| function | use | scene |
|------|------|------|
| `upload_file()` | Upload a single files | Update configuration and add files |
| `upload_folder()` | Upload the entire files folder | Completely upload models and data sets |

## Single files upload - upload_file()

### Basic usage

```python
from xiaoshiai_hub import upload_file

# 上传单个文件
upload_file(
    path_file="./config.json",
    path_in_repo="config.json",
    repo_id="my-org/my-model"
)

print("✓ 文件已上传")
```

> **Note**: The first two parameters of `upload_file` are `path_file` (local files path) and `path_in_repo` (path within repository), and `repo_id` is the third parameter.

### Complete parameters

```python
upload_file(
    path_file="./config.json",        # 本地文件路径（必需）
    path_in_repo="config.json",       # 仓库内的路径（必需）
    repo_id="my-org/my-model",        # 仓库 ID（必需）

    # 可选参数
    repo_type="models",               # "models" / "datasets" / "spaces"
    revision="main",                   # 分支名
    commit_message=None,               # 提交信息
    commit_description=None,           # 提交描述
    token=None,                        # API Token
    encryption_password=None,          # 加密密码
    algorithm=None,                    # 加密算法 "AES" / "SM4"
)
```

### Upload to subdirectory

```python
# 上传到仓库内的子目录
upload_file(
    path_file="./weights/model.safetensors",
    path_in_repo="weights/v1.0/model.safetensors",
    repo_id="my-org/my-model"
)
```

### Upload to datasetrepository

```python
upload_file(
    path_file="./train.json",
    path_in_repo="data/train.json",
    repo_id="my-org/my-dataset",
    repo_type="datasets"
)
```

## filesFolder upload - upload_folder()

### Basic usage

```python
from xiaoshiai_hub import upload_folder

# 上传整个文件夹
upload_folder(
    folder_path="./my_model",
    repo_id="my-org/my-model"
)

print("✓ 文件夹已上传")
```

### Ignore unnecessary files

`upload_folder` supports `ignore_patterns` parameter to skip files that does not need to be uploaded:

```python
# 忽略临时文件和缓存
upload_folder(
    folder_path="./my_model",
    repo_id="my-org/my-model",
    ignore_patterns=["*.pyc", "__pycache__", ".git", "*.tmp"]
)

# 实用示例：上传 PyTorch 项目，忽略环境和缓存
upload_folder(
    folder_path="./my_model",
    repo_id="my-org/my-model",
    ignore_patterns=[
        ".*",              # 隐藏文件
        "venv/*",          # 虚拟环境
        "__pycache__/*",   # Python 缓存
        "*.pyc",           # Python 编译文件
        "*.egg-info/*",    # 包信息
        "build/*",         # 构建目录
        "dist/*",          # 发行版目录
    ]
)
```

### Complete parameters

```python
upload_folder(
    folder_path="./my_model",         # 本地文件夹路径（必需）
    repo_id="my-org/my-model",        # 仓库 ID（必需）

    # 可选参数
    repo_type="models",               # "models" / "datasets" / "spaces"
    revision="main",                   # 分支名
    commit_message=None,               # 提交信息
    commit_description=None,           # 提交描述
    token=None,                        # API Token
    encryption_password=None,          # 加密密码
    ignore_patterns=None,              # 忽略的文件模式列表
    temp_dir=None,                     # 临时目录路径
    algorithm=None,                    # 加密算法 "AES" / "SM4"
)
```

## Actual combat scenario

### Scenario 1: Quickly download a single weight files

```python
from xiaoshiai_hub import moha_hub_download

# 只下载 safetensors 权重
weights_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="model.safetensors"
)

print(f"权重已下载: {weights_path}")
```

### Scenario 2: Download the model completely, skipping the large files

```python
from xiaoshiai_hub import snapshot_download

# 下载模型，但忽略 .bin 文件（因为已有 safetensors）
model_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns="*.bin"
)

# 目录结构
# model_path/
# ├── config.json
# ├── model.safetensors       ← 已下载
# ├── model.bin               ← 被忽略
# ├── tokenizer.model
# └── ...
```

### Scenario 3: Upload the model after fine-tuning

```python
from xiaoshiai_hub import upload_folder

# 微调完成后，上传模型（忽略不需要的文件）
upload_folder(
    folder_path="./output",
    repo_id="my-org/finetuned-qwen2",
    ignore_patterns=["*.pyc", "__pycache__", "*.bin"],
    commit_message="Finetuned model on custom dataset"
)

print("✓ 微调模型已上传")
```

### Scenario 4: Uploading large models in stages

```python
from xiaoshiai_hub import upload_file
import os

# 大型模型分为多个文件上传
model_files = [
    "model-00001-of-00003.safetensors",
    "model-00002-of-00003.safetensors",
    "model-00003-of-00003.safetensors",
    "model.safetensors.index.json"
]

for filename in model_files:
    file_path = os.path.join("./model", filename)
    if os.path.exists(file_path):
        upload_file(
            path_file=file_path,
            path_in_repo=filename,
            repo_id="my-org/large-model"
        )
        print(f"✓ 已上传 {filename}")
```

### Scenario 5: Encrypted upload

```python
from xiaoshiai_hub import upload_file, upload_folder

# 单文件加密上传
upload_file(
    path_file="./model.safetensors",
    path_in_repo="model.safetensors",
    repo_id="my-org/secret-model",
    encryption_password="my-secure-password",
    algorithm="AES"
)

# 文件夹加密上传
upload_folder(
    folder_path="./model",
    repo_id="my-org/secret-model",
    encryption_password="my-secure-password",
    algorithm="AES",
    ignore_patterns=["*.pyc", "__pycache__"]
)
```

> See [Encryption function](./07.encryption) for more encryption usage.

### Scenario 6: Download multiple datasets in batches

```python
from xiaoshiai_hub import snapshot_download

datasets = [
    "ai-lab/dataset-a",
    "ai-lab/dataset-b",
    "ai-lab/dataset-c"
]

for repo_id in datasets:
    path = snapshot_download(
        repo_id=repo_id,
        repo_type="datasets",
        local_dir=f"./data/{repo_id.split('/')[-1]}"
    )
    print(f"✓ {repo_id} 已下载到 {path}")
```

## Performance comparison

| scene | Recommended method | advantage |
|------|--------|------|
| Just one files | `moha_hub_download()` | Fast and takes up little space |
| Full model required | `snapshot_download()` + `allow_patterns` | Flexible, selective downloading |
| Upload a single files | `upload_file()` | Precise control |
| Upload entire directory | `upload_folder()` + `ignore_patterns` | Automatic processing, support filtering |
