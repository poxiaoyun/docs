## Overview

"Ecological Documents" is aimed at platform delivery and operation and maintenance, image production and adaptation engineers, and aggregates usage guides from different hardware ecosystems and open source communities.

You can quickly locate by source in the left directory:

- **Huawei (Ascend)**: driver/runtime, mirror environment, adaptation and tuning, troubleshooting
- **Hygon (DCU)**: driver/runtime, mirror environment, adaptation and tuning, troubleshooting
- **Other open source communities**: Kubernetes/containers, frameworks and inference ecology, observability, product security

:::info
This column gives priority to providing "version matrix + operation steps + known issues/FAQ". If you find a missing version or scene, please provide feedback through the team channel to complete it.
:::

## Directory description

- The document path corresponds to the directory structure one-to-one, for example: `/ecosystem/huawei/driver-runtime`
- It is recommended to follow the page: **Overview → Prerequisites → Operation Steps → Verification Method → FAQ**
