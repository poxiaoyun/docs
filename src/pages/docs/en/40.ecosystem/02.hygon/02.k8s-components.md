## illustrate

The content of this directory comes from the same-level document "DCU Kubernetes Component User Guide.pdf". To facilitate retrieval and maintenance, it has been split into multiple pages according to chapters.

- Overview: component introduction, image list, system requirements
- DCU-Label-Node: installation and deployment, deployment verification, node label description
- DCU-Exporter: indicator description, installation and deployment, verification, Grafana panel, physical machine deployment
- DCU-Device-Plugin: standard mode, MIG mode, vDCU dynamic segmentation mode
- FAQ: Frequently Asked Questions and Troubleshooting
