## Overview

Boss manages more than tenant workloads in a cluster. It also maintains shared storage services and platform runtime components that make logging, monitoring, development support, or system-level operations possible.

## Included Surfaces

- Use the storage page to inspect system storage instances and add new ones when required.
- Use built-in system images to review platform-level runtime components and their current status.
- Use the system market to deploy new shared capabilities into the cluster.
