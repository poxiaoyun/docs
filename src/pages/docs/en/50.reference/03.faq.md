## Overview

The FAQ section collects recurring questions that appear across onboarding, product usage, and operations. It is the best quick reference when a problem is common enough that you suspect someone has already solved it.

## When to Use It

- Use it after checking the product-specific guide for the feature you are working with.
- Use it when a symptom spans multiple modules and does not clearly belong to only one page.
- Treat repeated FAQ items as candidates for stronger runbooks or more explicit UI guidance.
