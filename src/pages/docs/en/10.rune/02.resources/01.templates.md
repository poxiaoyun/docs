## Overview

Instance templates are used to turn a working deployment into a reusable asset. They are especially helpful when a team wants to standardize model paths, runtime options, storage mounts, and parameter defaults across repeated jobs.

## Common Actions

- Save an existing instance as a reusable template once the parameters are stable.
- Launch new deployments from the template instead of rebuilding the full form each time.
- Use the form view for guided editing and the raw JSON view for advanced adjustments.
- Share templates with teammates who work inside the same tenant or workspace boundary.
