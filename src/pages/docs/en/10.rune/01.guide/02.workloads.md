## Overview

Inference services, fine-tuning jobs, dev environments, applications, experiments, and evaluations all follow the same basic deployment shape in Rune: select a template, fill parameters, choose a flavor, and submit an instance.

## Shared Workflow

- Start from the App Market or a template entry point.
- Fill in instance metadata and runtime parameters in the generated form.
- Choose a compute flavor that fits the workload and the current quota boundary.
- Submit the deployment, then follow status, logs, events, and monitoring from the instance detail page.
