## Overview

The Rune dashboard is the landing view for the selected cluster and workspace. It helps users confirm that they are in the right context, understand the available resource boundary, and spot active or abnormal workloads before drilling down.

## Typical Uses

- Check current workspace and cluster context before creating a new instance.
- Review high-level counters for running inference services, jobs, apps, or development environments.
- See quota usage trends and decide whether to stop or resize workloads.
- Use the dashboard as the starting point before opening workload lists, logs, or monitoring pages.
