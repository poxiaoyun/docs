## 快速开始

帮助您快速上手魔哈仓库的核心功能。

## 准备工作

使用魔哈仓库前，请确认：

1. **拥有平台账户** — 联系管理员创建账户或自行注册
2. **加入组织** — 模型和数据集归属于组织，需先加入或创建组织
3. **创建访问令牌** — 用于 Git 操作和 API 调用的身份认证

## 快速上手流程

```mermaid
flowchart LR
    A["注册/登录"] --> B["加入组织"]
    B --> C["创建令牌"]
    C --> D["上传模型/数据集"]
    D --> E["协作分享"]
```
