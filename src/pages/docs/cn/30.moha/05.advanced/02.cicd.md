## 概述

集成持续训练和部署流程，实现自动化 MLOps。

## GitHub Actions 集成

### 自动训练

```yaml
name: Train Model

on:
  push:
    branches: [main]

jobs:
  train:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      
      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: '3.9'
      
      - name: Install dependencies
        run: |
          pip install -r requirements.txt
      
      - name: Train model
        run: |
          python train.py
      
      - name: Upload to Moha
        env:
          MOHA_API_KEY: ${{ secrets.MOHA_API_KEY }}
        run: |
          moha-cli upload --model-path ./model.pt --version ${{ github.sha }}
```

### 自动测试

```yaml
- name: Test model
  run: |
    python test.py
    
- name: Upload test results
  if: success()
  run: |
    moha-cli test-report --results ./test_results.json
```

## GitLab CI 集成

```yaml
stages:
  - train
  - test
  - deploy

train_model:
  stage: train
  script:
    - python train.py
    - moha-cli upload --model-path ./model.pt

test_model:
  stage: test
  script:
    - python test.py

deploy_model:
  stage: deploy
  script:
    - moha-cli deploy --model-id $MODEL_ID
```
