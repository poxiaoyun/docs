# 镜像仓库

镜像仓库用于管理可复用的运行镜像资源，适合沉淀训练、推理和 Space 运行所需的基础环境。

## 进入路径

魔哈顶部导航 -> 镜像

路径：`/moha/images`

## 页面能力

- 支持公开与私有视图切换。
- 支持搜索、排序和筛选。
- 使用统一的数据卡片展示镜像信息。
- 进入仓库详情后可查看标签、README 和设置。
