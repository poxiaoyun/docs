# 镜像列表与筛选

镜像列表页沿用魔哈统一列表框架，适合从资产发现和镜像治理两个角度使用。

## 页面特征

| 能力 | 说明 |
| --- | --- |
| 可见性切换 | 在公开、私有或内部资源之间切换 |
| 搜索 | 通过名称或关键字定位镜像 |
| 排序 | 按下载量、点赞等指标排序 |
| 数据卡片 | 查看镜像基础信息并进入详情页 |

## 使用建议

- 面向全平台复用的公共基础镜像，建议维护清晰的说明和标签。
- 若镜像只服务于特定团队，可采用私有或内部可见性。
- 需要维护镜像元数据时，进入仓库设置页更新标签、别名和 README。
