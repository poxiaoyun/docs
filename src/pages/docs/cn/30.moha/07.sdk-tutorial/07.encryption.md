## 加密概述

Moha SDK 支持基于密码的文件加密上传，核心能力包括：

- 上传时对文件内容进行加密
- 支持 `AES` 和 `SM4` 两种算法
- 下载时自动识别加密文件头并解密（需提供正确密码）
- 支持通过 `MOHA_ENCRYPTION_PASSWORD` 环境变量提供默认密码

### 算法说明

| 算法 | 说明 | 适用场景 |
|------|------|----------|
| `AES` | 国际通用，性能较好 | 默认推荐 |
| `SM4` | 国密算法 | 需国密合规场景 |

## CLI 加密上传

### 上传文件夹并加密

```bash
moha upload \
  -t models \
  -e \
  --encryption-password "your-password" \
  -a AES \
  -m "Encrypted upload" \
  "my-org/my-model" \
  ./model
```

### 上传单文件并加密

```bash
moha upload-file \
  -t models \
  -e \
  --encryption-password "your-password" \
  -a SM4 \
  -m "Upload encrypted weight" \
  "my-org/my-model" \
  ./model.safetensors \
  model.safetensors
```

### 使用环境变量提供密码

```bash
export MOHA_ENCRYPTION_PASSWORD="your-password"

moha upload \
  -t datasets \
  -e \
  -a AES \
  "my-org/my-dataset" \
  ./dataset
```

> `-e/--encrypt` 仅开启加密流程；密码来自 `--encryption-password` 或 `MOHA_ENCRYPTION_PASSWORD`。

## Python API 加密上传

### 单文件加密上传

```python
from xiaoshiai_hub import upload_file

upload_file(
    path_file="./model.safetensors",
    path_in_repo="weights/model.safetensors",
    repo_id="my-org/my-model",
    repo_type="models",
    encryption_password="your-password",
    algorithm="AES",
    commit_message="Upload encrypted model weight"
)

print("✓ 文件已加密上传")
```

### 文件夹加密上传

```python
from xiaoshiai_hub import upload_folder

upload_folder(
    folder_path="./dataset",
    repo_id="my-org/my-dataset",
    repo_type="datasets",
    encryption_password="your-password",
    algorithm="SM4",
    ignore_patterns=["*.tmp", "__pycache__", "*.pyc"],
    commit_message="Upload encrypted dataset"
)

print("✓ 文件夹已加密上传")
```

### 关键参数

| 参数 | 作用 |
|------|------|
| `encryption_password` | 加密密码 |
| `algorithm` | 加密算法，支持 `"AES"` / `"SM4"` |
| `ignore_patterns` | 上传文件夹时忽略文件模式 |

## 下载与解密

加密文件下载时，SDK 会根据文件头自动识别并执行解密。你需要在运行环境中提供正确密码。

### CLI 下载

```bash
export MOHA_ENCRYPTION_PASSWORD="your-password"

moha download \
  -t models \
  "my-org/my-model" \
  -o ./downloads
```

### Python 下载

```python
import os
from xiaoshiai_hub import snapshot_download, moha_hub_download

os.environ["MOHA_ENCRYPTION_PASSWORD"] = "your-password"

# 下载整个仓库
repo_path = snapshot_download(
    repo_id="my-org/my-model",
    repo_type="models",
    local_dir="./downloads/model"
)
print(f"✓ 仓库已下载: {repo_path}")

# 下载单个文件
file_path = moha_hub_download(
    repo_id="my-org/my-model",
    filename="weights/model.safetensors",
    repo_type="models",
    local_dir="./downloads/single"
)
print(f"✓ 文件已下载: {file_path}")
```

## DataKey 与 KMS

`HubClient` 提供 `generate_data_key()` 用于向服务端请求数据密钥：

```python
from xiaoshiai_hub import HubClient

client = HubClient(token="sk_xxxxx")

data_key = client.generate_data_key(
    algorithm="AES",
    password="your-password"
)

print(type(data_key).__name__)  # DataKey
print(len(data_key.plaintext_key))
print(data_key.encrypted_key[:24] + "...")
```

`DataKey` 是简单数据结构，字段如下：

```python
@dataclass
class DataKey:
    plaintext_key: bytes
    encrypted_key: str
```

> 当前 SDK 中，`DataKey` 没有 `generate()` / `save()` / `from_file()` / `revoke()` 等方法。

## 实战工作流

### 场景 1：上传敏感数据集

```python
from xiaoshiai_hub import upload_folder

upload_folder(
    folder_path="./sensitive_dataset",
    repo_id="my-org/sensitive-data",
    repo_type="datasets",
    encryption_password="team-secret",
    algorithm="AES",
    ignore_patterns=["*.tmp", "*.log", "__pycache__"],
    commit_message="Encrypted dataset upload"
)

print("✓ 敏感数据集已加密上传")
```

### 场景 2：团队共享加密模型

```python
import os
from xiaoshiai_hub import upload_file, snapshot_download

# 管理员上传（加密）
upload_file(
    path_file="./model.safetensors",
    path_in_repo="model.safetensors",
    repo_id="my-team/shared-model",
    encryption_password="team-password",
    algorithm="AES",
    commit_message="Upload encrypted team model"

    repo_id="my-team/shared-model",
    repo_type="models"
)
print(f"✓ 团队模型已下载: {model_path}")
```

## 安全建议

### 密码管理

- 不要把加密密码硬编码到仓库代码中。
- 优先使用环境变量或 CI Secret 注入。
- 不同项目建议使用不同密码，降低横向风险。

### CI/CD 示例（GitHub Actions）

```yaml
name: Upload Encrypted Artifacts

on:
  push:
    branches: [main]

jobs:
  upload:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.10'

      - name: Install SDK
        run: pip install xiaoshiai-hub

      - name: Upload encrypted folder
        env:
          MOHA_TOKEN: ${{ secrets.MOHA_TOKEN }}
          MOHA_ENCRYPTION_PASSWORD: ${{ secrets.MOHA_ENCRYPTION_PASSWORD }}
        run: |
          moha upload \
            -t models \
            -e \
            -a AES \
            -m "CI encrypted upload" \
            my-org/my-model \
            ./model
```

## 故障排查

### 1. 上传时报错：缺少加密密码

症状：启用了 `-e` 或设置了加密参数，但没有提供密码。

解决：

```bash
export MOHA_ENCRYPTION_PASSWORD="your-password"
# 或在命令中显式传 --encryption-password
```

### 2. 下载后解密失败

症状：下载成功但无法解密内容。

解决：

- 检查 `MOHA_ENCRYPTION_PASSWORD` 是否正确。
- 确认上传和下载使用的是同一密码。
- 确认算法和文件未被篡改。

### 3. 算法参数错误

症状：传入了不支持的算法字符串。

解决：仅使用 `AES` 或 `SM4`。
