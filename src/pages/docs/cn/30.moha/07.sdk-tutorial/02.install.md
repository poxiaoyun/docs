## 安装

### 基础安装

最简单的安装方式：

```bash
pip install xiaoshiai-hub
```

### 安装特定版本

```bash
# 安装指定版本（例如 1.2.0）
pip install xiaoshiai-hub==1.2.0

# 安装最新开发版本（来自 GitHub）
pip install git+https://github.com/your-org/moha-sdk.git
```

## 依赖要求

### Python 版本

- **最低要求**：Python 3.7+
- **推荐版本**：Python 3.9+ （更好的性能和兼容性）

### 核心依赖

SDK 在安装时会自动安装以下依赖：

| 依赖 | 版本范围 | 用途 |
|-----|--------|------|
| `requests` | >=2.20.0, <3.0.0 | HTTP 请求库 |
| `tqdm` | >=4.62.0, <5.0.0 | 进度条显示 |
| `cryptography` | >=46.0.3, <47.0.0 | 加密/解密操作 |

### 可选依赖

#### 上传功能 - Git 支持

如果要使用 `upload_folder` 或基于 Git 的操作，需要安装：

```bash
pip install xiaoshiai-hub[upload]
```

这将额外安装：
- `gitpython>=3.1.0,<4.0.0` — Git 操作库

#### 加密功能 - 企业级加密

SDK 内置 AES-256-CTR 加密，但企业级的 SM4-CTR 加密需要外部库：

```bash
# 注意：xpai-enc 不在 PyPI，需从 GitHub 安装
pip install git+https://github.com/poxiaoyun/xpai-enc.git
```

### 开发依赖（贡献者）

如果要参与开发或运行测试：

```bash
pip install xiaoshiai-hub[dev]
```

## 环境变量配置

### 基本环境变量

SDK 通过以下环境变量进行配置：

| 环境变量 | 说明 | 示例 | 优先级 |
|---------|------|------|--------|
| `MOHA_ENDPOINT` | Moha Hub 服务地址 | `https://moha.your-domain` | CLI 中最高优先级 |
| `MOHA_TOKEN` | API Token（推荐） | `sk_xxxxxxxxxxxx` | 1 |
| `MOHA_USERNAME` | 用户名 | `your_username` | 2 |
| `MOHA_PASSWORD` | 密码 | `your_password` | 2 |

### 设置环境变量

#### Linux / macOS

```bash
# 临时设置（仅当前 shell 会话）
export MOHA_ENDPOINT=https://moha.your-domain
export MOHA_TOKEN=your_token

# 永久设置（添加到 ~/.bashrc 或 ~/.zshrc）
echo 'export MOHA_ENDPOINT=https://moha.your-domain' >> ~/.bashrc
source ~/.bashrc
```

#### Windows

```powershell
# 临时设置（仅当前 PowerShell 会话）
$env:MOHA_ENDPOINT = "https://moha.your-domain"
$env:MOHA_TOKEN = "your_token"

# 永久设置
[Environment]::SetEnvironmentVariable("MOHA_ENDPOINT", "https://moha.your-domain", "User")
```

### 在 Python 代码中配置

如果不想使用环境变量，可以在代码中直接设置：

```python
import os

# 设置环境变量
os.environ["MOHA_ENDPOINT"] = "https://moha.your-domain"
os.environ["MOHA_TOKEN"] = "your_token"

# 或者直接在初始化时传参
from xiaoshiai_hub import HubClient

client = HubClient(
    base_url="https://moha.your-domain",
    token="your_token"
)
```

## 验证安装

### 检查 SDK 版本

```bash
python -c "from xiaoshiai_hub import __version__; print(__version__)"
```

### 测试连接

```python
from xiaoshiai_hub import HubClient

client = HubClient(base_url="https://moha.your-domain")
models = client.list_repositories("models")
print(f"✓ 连接成功，获取到 {len(models)} 个模型")
```

### 测试认证

```python
from xiaoshiai_hub import load_token

try:
    token, base_url, username = load_token()
    print(f"✓ 认证成功，用户: {username}")
except Exception as e:
    print(f"✗ 认证失败: {e}")
```

## 离线部署

### 下载依赖到本地

在有网络的机器上：

```bash
pip download xiaoshiai-hub -d ./wheels
```

### 在离线环境安装

将 `wheels` 目录复制到离线机器，然后：

```bash
pip install --no-index --find-links ./wheels xiaoshiai-hub
```

## 常见安装问题

### 问题 1：`pip install` 超时

**症状**：安装过程中网络超时。

**解决**：
```bash
# 使用国内镜像源
pip install -i https://pypi.tsinghua.edu.cn/simple xiaoshiai-hub

# 或增加超时时间
pip install --default-timeout=1000 xiaoshiai-hub
```

### 问题 2：`cryptography` 编译失败

**症状**：
```
error: Microsoft Visual C++ 14.0 or greater is required
```

**解决（Windows）**：
- 下载安装 [Microsoft C++ Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
- 或安装预编译的 wheel：
  ```bash
  pip install --only-binary :all: cryptography
  ```

### 问题 3：权限拒绝（Permission Denied）

**症状**：
```
ERROR: Could not install packages due to an EnvironmentError: [Errno 13]
```

**解决**：
```bash
# 方式 1：使用 --user flag（推荐）
pip install --user xiaoshiai-hub

# 方式 2：使用虚拟环境（最佳实践）
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或 venv\Scripts\activate  # Windows
pip install xiaoshiai-hub
```

### 问题 4：`xpai-enc` 安装失败（仅使用加密时需要）

**症状**：
```
ERROR: Could not find a version that satisfies the requirement xpai-enc
```

**解决**：
```bash
# 使用 GitHub 安装
pip install git+https://github.com/poxiaoyun/xpai-enc.git

# 如果没有 git，下载源码手动安装
git clone https://github.com/poxiaoyun/xpai-enc.git
cd xpai-enc
pip install .
```

## 虚拟环境设置（推荐）

### 使用 venv

```bash
# 创建虚拟环境
python -m venv moha-env

# 激活虚拟环境
source moha-env/bin/activate  # Linux/macOS
# 或
moha-env\Scripts\activate  # Windows

# 安装 SDK
pip install xiaoshiai-hub

# 使用完后退出虚拟环境
deactivate
```

### 使用 conda

```bash
# 创建 conda 环境
conda create -n moha-env python=3.9

# 激活环境
conda activate moha-env

# 安装 SDK
pip install xiaoshiai-hub

# 退出环境
conda deactivate
```
