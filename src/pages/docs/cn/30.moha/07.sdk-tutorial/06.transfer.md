## 下载概述

Moha SDK 提供两种下载方式：

| 函数 | 用途 | 场景 |
|------|------|------|
| `moha_hub_download()` | 下载单个文件 | 快速加载单个模型文件 |
| `snapshot_download()` | 下载整个仓库快照 | 完整下载数据集或模型 |

## 单文件下载 - moha_hub_download()

### 基础用法

```python
from xiaoshiai_hub import moha_hub_download

# 下载单个文件
config_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json"
)

print(f"文件位置: {config_path}")
```

### 参数详解

```python
moha_hub_download(
    repo_id="ai-lab/qwen2-7b",    # 仓库 ID (org/name)
    filename="config.json",         # 文件名

    # 可选参数
    repo_type="models",             # "models" / "datasets" / "spaces"
    revision="main",                # 分支/标签名
    local_dir="./models",           # 下载到本地文件夹
    base_url=None,                  # 自定义 Hub 地址
    token=None,                     # API Token
    show_progress=True,             # 显示进度条
)
```

**参数说明**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `repo_id` | `str` | — | 仓库 ID，格式为 `org/name` |
| `filename` | `str` | — | 要下载的文件名 |
| `repo_type` | `str` | `"models"` | 仓库类型 |
| `revision` | `str` | `None` | 分支名或标签名 |
| `local_dir` | `str` | `None` | 本地保存目录 |
| `show_progress` | `bool` | `True` | 是否显示进度条 |

### 指定本地保存位置

```python
# 保存到特定目录
config_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    local_dir="./my_models"
)
# 保存到 ./my_models/config.json

# 下载模型权重文件
weights_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="model.safetensors",
    local_dir="./models"
)
# 保存到 ./models/model.safetensors
```

### 指定版本下载

```python
# 下载特定分支
config = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    revision="dev"
)

# 下载特定标签
v1 = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    revision="v1.0"
)
```

### 静默下载（不显示进度条）

```python
config_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="config.json",
    show_progress=False
)
```

## 完整仓库下载 - snapshot_download()

### 基础用法

```python
from xiaoshiai_hub import snapshot_download

# 下载整个模型仓库
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b"
)

print(f"仓库位置: {repo_path}")
# 目录结构
# {repo_path}/
# ├── config.json
# ├── model.safetensors
# ├── tokenizer.model
# └── ...
```

### 参数详解

```python
snapshot_download(
    repo_id="ai-lab/qwen2-7b",     # 仓库 ID (org/name)

    # 可选参数
    repo_type="models",              # "models" / "datasets" / "spaces"
    revision=None,                   # 分支/标签名
    local_dir=None,                  # 本地保存目录
    allow_patterns=None,             # 只下载匹配的文件
    ignore_patterns=None,            # 忽略匹配的文件
    base_url=None,                   # 自定义 Hub 地址
    token=None,                      # API Token
    show_progress=True,              # 显示进度条
)
```

### 选择性下载（模式过滤）

#### allow_patterns - 只下载匹配文件

```python
# 只下载 safetensors 权重文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns="*.safetensors"
)

# 下载多种文件格式
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns=["*.safetensors", "*.json", "*.model"]
)

# 下载特定目录
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns="weights/*"
)
```

#### ignore_patterns - 忽略匹配文件

```python
# 下载所有文件，但忽略大文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns="*.bin"
)

# 忽略多种文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns=["*.pyc", "__pycache__", "*.tmp"]
)

# 忽略特定目录
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns=["build/*", "test/*"]
)
```

#### 组合使用

```python
# 只下载权重和配置，忽略临时文件
repo_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    allow_patterns=["*.safetensors", "*.json"],
    ignore_patterns=["*.tmp"]
)
```

## 上传概述

Moha SDK 提供两种上传方式：

| 函数 | 用途 | 场景 |
|------|------|------|
| `upload_file()` | 上传单个文件 | 更新配置、添加文件 |
| `upload_folder()` | 上传整个文件夹 | 完整上传模型、数据集 |

## 单文件上传 - upload_file()

### 基础用法

```python
from xiaoshiai_hub import upload_file

# 上传单个文件
upload_file(
    path_file="./config.json",
    path_in_repo="config.json",
    repo_id="my-org/my-model"
)

print("✓ 文件已上传")
```

> **注意**：`upload_file` 的前两个参数是 `path_file`（本地文件路径）和 `path_in_repo`（仓库内路径），`repo_id` 是第三个参数。

### 完整参数

```python
upload_file(
    path_file="./config.json",        # 本地文件路径（必需）
    path_in_repo="config.json",       # 仓库内的路径（必需）
    repo_id="my-org/my-model",        # 仓库 ID（必需）

    # 可选参数
    repo_type="models",               # "models" / "datasets" / "spaces"
    revision="main",                   # 分支名
    commit_message=None,               # 提交信息
    commit_description=None,           # 提交描述
    token=None,                        # API Token
    encryption_password=None,          # 加密密码
    algorithm=None,                    # 加密算法 "AES" / "SM4"
)
```

### 上传到子目录

```python
# 上传到仓库内的子目录
upload_file(
    path_file="./weights/model.safetensors",
    path_in_repo="weights/v1.0/model.safetensors",
    repo_id="my-org/my-model"
)
```

### 上传到数据集仓库

```python
upload_file(
    path_file="./train.json",
    path_in_repo="data/train.json",
    repo_id="my-org/my-dataset",
    repo_type="datasets"
)
```

## 文件夹上传 - upload_folder()

### 基础用法

```python
from xiaoshiai_hub import upload_folder

# 上传整个文件夹
upload_folder(
    folder_path="./my_model",
    repo_id="my-org/my-model"
)

print("✓ 文件夹已上传")
```

### 忽略不需要的文件

`upload_folder` 支持 `ignore_patterns` 参数来跳过不需要上传的文件：

```python
# 忽略临时文件和缓存
upload_folder(
    folder_path="./my_model",
    repo_id="my-org/my-model",
    ignore_patterns=["*.pyc", "__pycache__", ".git", "*.tmp"]
)

# 实用示例：上传 PyTorch 项目，忽略环境和缓存
upload_folder(
    folder_path="./my_model",
    repo_id="my-org/my-model",
    ignore_patterns=[
        ".*",              # 隐藏文件
        "venv/*",          # 虚拟环境
        "__pycache__/*",   # Python 缓存
        "*.pyc",           # Python 编译文件
        "*.egg-info/*",    # 包信息
        "build/*",         # 构建目录
        "dist/*",          # 发行版目录
    ]
)
```

### 完整参数

```python
upload_folder(
    folder_path="./my_model",         # 本地文件夹路径（必需）
    repo_id="my-org/my-model",        # 仓库 ID（必需）

    # 可选参数
    repo_type="models",               # "models" / "datasets" / "spaces"
    revision="main",                   # 分支名
    commit_message=None,               # 提交信息
    commit_description=None,           # 提交描述
    token=None,                        # API Token
    encryption_password=None,          # 加密密码
    ignore_patterns=None,              # 忽略的文件模式列表
    temp_dir=None,                     # 临时目录路径
    algorithm=None,                    # 加密算法 "AES" / "SM4"
)
```

## 实战场景

### 场景 1：快速下载单个权重文件

```python
from xiaoshiai_hub import moha_hub_download

# 只下载 safetensors 权重
weights_path = moha_hub_download(
    repo_id="ai-lab/qwen2-7b",
    filename="model.safetensors"
)

print(f"权重已下载: {weights_path}")
```

### 场景 2：完整下载模型，跳过大文件

```python
from xiaoshiai_hub import snapshot_download

# 下载模型，但忽略 .bin 文件（因为已有 safetensors）
model_path = snapshot_download(
    repo_id="ai-lab/qwen2-7b",
    ignore_patterns="*.bin"
)

# 目录结构
# model_path/
# ├── config.json
# ├── model.safetensors       ← 已下载
# ├── model.bin               ← 被忽略
# ├── tokenizer.model
# └── ...
```

### 场景 3：微调后上传模型

```python
from xiaoshiai_hub import upload_folder

# 微调完成后，上传模型（忽略不需要的文件）
upload_folder(
    folder_path="./output",
    repo_id="my-org/finetuned-qwen2",
    ignore_patterns=["*.pyc", "__pycache__", "*.bin"],
    commit_message="Finetuned model on custom dataset"
)

print("✓ 微调模型已上传")
```

### 场景 4：分阶段上传大型模型

```python
from xiaoshiai_hub import upload_file
import os

# 大型模型分为多个文件上传
model_files = [
    "model-00001-of-00003.safetensors",
    "model-00002-of-00003.safetensors",
    "model-00003-of-00003.safetensors",
    "model.safetensors.index.json"
]

for filename in model_files:
    file_path = os.path.join("./model", filename)
    if os.path.exists(file_path):
        upload_file(
            path_file=file_path,
            path_in_repo=filename,
            repo_id="my-org/large-model"
        )
        print(f"✓ 已上传 {filename}")
```

### 场景 5：加密上传

```python
from xiaoshiai_hub import upload_file, upload_folder

# 单文件加密上传
upload_file(
    path_file="./model.safetensors",
    path_in_repo="model.safetensors",
    repo_id="my-org/secret-model",
    encryption_password="my-secure-password",
    algorithm="AES"
)

# 文件夹加密上传
upload_folder(
    folder_path="./model",
    repo_id="my-org/secret-model",
    encryption_password="my-secure-password",
    algorithm="AES",
    ignore_patterns=["*.pyc", "__pycache__"]
)
```

> 更多加密用法参见 [加密功能](./07.encryption)。

### 场景 6：批量下载多个数据集

```python
from xiaoshiai_hub import snapshot_download

datasets = [
    "ai-lab/dataset-a",
    "ai-lab/dataset-b",
    "ai-lab/dataset-c"
]

for repo_id in datasets:
    path = snapshot_download(
        repo_id=repo_id,
        repo_type="datasets",
        local_dir=f"./data/{repo_id.split('/')[-1]}"
    )
    print(f"✓ {repo_id} 已下载到 {path}")
```

## 性能对比

| 场景 | 推荐方法 | 优点 |
|------|--------|------|
| 只需一个文件 | `moha_hub_download()` | 快速、占用空间小 |
| 需要完整模型 | `snapshot_download()` + `allow_patterns` | 灵活、选择性下载 |
| 上传单个文件 | `upload_file()` | 精确控制 |
| 上传整个目录 | `upload_folder()` + `ignore_patterns` | 自动处理、支持过滤 |
