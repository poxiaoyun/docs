## HubClient 概述

`HubClient` 是 Moha SDK 的核心类，提供直接调用 Hub API 的方式。适合需要编程灵活性和细粒度控制的场景。

### 初始化

```python
from xiaoshiai_hub import HubClient

# 基础初始化（使用环境变量或本地 Token）
client = HubClient()

# 完整初始化
client = HubClient(
    base_url="https://moha.your-domain",
    token="sk_xxxxxxxxxxxx"
)

# 使用用户名/密码
client = HubClient(
    base_url="https://moha.your-domain",
    username="alice",
    password="secret"
)
```

**参数说明**：

| 参数 | 类型 | 说明 |
|------|------|------|
| `base_url` | `str` | Hub API 地址（默认从 `MOHA_ENDPOINT` 环境变量获取） |
| `username` | `str` | 用户名 |
| `password` | `str` | 密码 |
| `token` | `str` | API Token（与用户名/密码二选一） |

## 仓库操作

### 列出仓库 - `list_repositories()`

列出仓库，支持按类型、范围、组织过滤。

```python
# 列出公开模型
models = client.list_repositories(repo_type="models")
for model in models:
    print(f"{model.organization}/{model.name} - {model.description}")

# 列出公开数据集
datasets = client.list_repositories(repo_type="datasets")

# 列出我创建的模型
my_models = client.list_repositories(repo_type="models", scope="create")

# 列出我收藏的模型
fav_models = client.list_repositories(repo_type="models", scope="favorite")

# 列出组织内的模型
org_models = client.list_repositories(
    repo_type="models",
    scope="organization",
    organization="my-org"
)

# 列出组织内我创建的模型
mine = client.list_repositories(
    repo_type="models",
    scope="organization",
    organization="my-org",
    mycreated=True
)
```

**参数**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `repo_type` | `str` | `"models"` | 仓库类型：`"models"` / `"datasets"` / `"spaces"` |
| `scope` | `str` | `"public"` | 查询范围：`"public"` / `"create"` / `"favorite"` / `"organization"` |
| `organization` | `str` | `None` | 组织名称（`scope="organization"` 时必需） |
| `mycreated` | `bool` | `False` | 仅显示我创建的（仅 `scope="organization"` 时有效） |

**返回**：`List[Repository]`

### 获取仓库信息 - `get_repository_info()`

```python
info = client.get_repository_info(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b"
)

print(f"名称: {info.name}")
print(f"组织: {info.organization}")
print(f"类型: {info.type}")
print(f"可见性: {info.visibility}")
print(f"描述: {info.description}")
```

**参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| `organization` | `str` | 组织名称 |
| `repo_type` | `str` | 仓库类型：`"models"` / `"datasets"` / `"spaces"` |
| `repo_name` | `str` | 仓库名称 |

**返回**：`Repository`

### 创建仓库 - `create_repository()`

```python
client.create_repository(
    organization="my-org",
    repo_type="models",
    repo_name="my-new-model",
    description="My awesome model",
    visibility="internal",
    metadata={
        "license": ["apache-2.0"],
        "tasks": ["text-generation"],
        "frameworks": ["pytorch"],
    },
    base_model=["ai-lab/qwen2-7b"],
    relationship="finetune"
)
print("✓ 仓库创建成功")
```

**参数**：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `organization` | `str` | — | 组织名称 |
| `repo_type` | `str` | — | 仓库类型 |
| `repo_name` | `str` | — | 仓库名称 |
| `description` | `str` | `None` | 仓库描述 |
| `visibility` | `str` | `"internal"` | 可见性：`"public"` / `"internal"` / `"private"` |
| `metadata` | `dict` | `None` | 元数据（license, tasks, languages, tags, frameworks） |
| `base_model` | `list` | `None` | 基础模型列表 |
| `relationship` | `str` | `None` | 与基础模型关系 |

### 更新仓库 - `update_repository()`

## 仓库内容操作

### 获取仓库内容 - `get_repository_content()`

获取仓库在指定路径下的文件/目录结构。

```python
# 获取仓库根目录内容
content = client.get_repository_content(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b",
    branch="main"
)

# 遍历根目录下的内容
if content.entries:
    for item in content.entries:
        item_type = "📁" if item.type == "dir" else "📄"
        print(f"{item_type} {item.path} ({item.size} bytes)")

# 获取子目录内容
sub_content = client.get_repository_content(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b",
    branch="main",
    path="weights"
)
```

**返回**：`GitContent` — 包含 `name`, `path`, `type`, `size`, `entries` 等字段

### 创建/更新文件 - `create_file()`

适用于小型文本文件（如配置文件、README 等）。

```python
client.create_file(
    organization="my-org",
    repo_type="models",
    repo_name="my-model",
    branch="main",
    file_path="README.md",
    content="# My Model\n\nThis is my fine-tuned model.",
    message="Add README"
)
print("✓ 文件已创建")
```

### 下载文件 - `download_file()`

下载仓库中的单个文件到本地。

```python
client.download_file(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b",
    branch="main",
    file_path="config.json",
    local_path="./downloaded_config.json",
    show_progress=True
)
print("✓ 文件已下载")
```

### 删除文件 - `delete_file()`

```python
client.delete_file(
    organization="my-org",
    repo_type="models",
    repo_name="my-model",
    branch="main",
    file_path="old_file.txt",
    message="Remove outdated file"
)
print("✓ 文件已删除")
```

## 版本管理

### 获取分支和标签 - `get_repository_refs()`

```python
refs = client.get_repository_refs(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b"
)

for ref in refs:
    ref_type = "分支" if ref.type == "branch" else "标签"
    default_mark = " (默认)" if ref.is_default else ""
    print(f"[{ref_type}] {ref.name}{default_mark} → {ref.hash[:8]}")
```

**返回**：`List[Ref]` — 每个 `Ref` 包含 `name`, `ref`, `type`, `hash`, `is_default` 字段

### 获取默认分支 - `get_default_branch()`

```python
default = client.get_default_branch(
    organization="ai-lab",
    repo_type="models",
    repo_name="qwen2-7b"
)
print(f"默认分支: {default}")  # 通常是 "main"
```

### 创建分支 - `create_branch()`

```python
client.create_branch(
```

### 删除分支 - `delete_branch()`

## 加密操作

### 生成数据密钥 - `generate_data_key()`

通过 KMS 服务生成数据密钥，用于信封加密。

```python
from xiaoshiai_hub import HubClient

client = HubClient(token="sk_xxxx")
data_key = client.generate_data_key(
    algorithm="AES",            # "AES" 或 "SM4"
    password="your_password"
)

print(f"明文密钥长度: {len(data_key.plaintext_key)} 字节")
print(f"加密密钥: {data_key.encrypted_key[:20]}...")
```

### 设置仓库加密标志 - `set_repository_encrypted()`

```python
client.set_repository_encrypted(
    organization="my-org",
    repo_type="models",
    repo_name="encrypted-model"
)
```

### 取消仓库加密标志 - `cancel_repository_encrypted()`

```python
client.cancel_repository_encrypted(
    organization="my-org",
    repo_type="models",
    repo_name="encrypted-model"
)
```

## 高级用法

### 完整工作流示例

```python
from xiaoshiai_hub import HubClient

# 1. 初始化客户端
client = HubClient(token="sk_xxxxxxxxxxxx")

# 2. 创建新仓库
client.create_repository(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model",
    description="Fine-tuned model",
    visibility="internal"
)
print("✓ 仓库创建成功")

# 3. 创建 README
client.create_file(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model",
    branch="main",
    file_path="README.md",
    content="# Finetuned Model\n\nFine-tuned on custom dataset.",
    message="Add README"
)
print("✓ README 已创建")

# 4. 创建开发分支
client.create_branch(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model",
    branch_name="dev",
    from_branch="main"
)
print("✓ dev 分支已创建")

# 5. 获取仓库信息
info = client.get_repository_info(
    organization="my-org",
    repo_type="models",
    repo_name="finetuned-model"
)
print(f"✓ 仓库: {info.organization}/{info.name}, 可见性: {info.visibility}")
```

### 批量操作

```python
# 批量下载仓库中的多个文件
import os

files_to_download = ["config.json", "tokenizer.model", "README.md"]

for filename in files_to_download:
    try:
        local_path = os.path.join("./downloads", filename)
        os.makedirs(os.path.dirname(local_path), exist_ok=True)
        client.download_file(
            organization="ai-lab",
            repo_type="models",
            repo_name="qwen2-7b",
            branch="main",
            file_path=filename,
            local_path=local_path
        )
        print(f"✓ 已下载: {filename}")
    except Exception as e:
        print(f"✗ 下载失败: {filename} - {e}")
```

### 错误处理集成

```python
from xiaoshiai_hub import (
    HubClient,
    RepositoryNotFoundError,
    AuthenticationError,
    HTTPError
)

client = HubClient()

try:
    info = client.get_repository_info(
        organization="ai-lab",
        repo_type="models",
        repo_name="non-existent"
    )
except RepositoryNotFoundError:
    print("仓库不存在")
except AuthenticationError:
    print("认证失败，请检查 Token")
except HTTPError as e:
    print(f"HTTP 错误: {e}")
```

## 数据模型参考

### Repository

```python
@dataclass
class Repository:
    name: str              # 仓库名称
    organization: str      # 组织名称
    owner: str             # 所有者
    creator: str           # 创建者
    type: str              # 类型: "models" / "datasets" / "spaces"
    visibility: str        # 可见性: "public" / "internal" / "private"
    description: str       # 描述
    metadata: dict         # 元数据 (license, tasks, languages 等)
    annotations: dict      # 注解
    genealogy: dict        # 模型谱系信息
    spaceMetadata: SpaceMetadata  # Space 元数据（仅 spaces）
```

### Ref

```python
@dataclass
class Ref:
    name: str         # 分支/标签名称
    ref: str          # 引用路径
    type: str         # "branch" 或 "tag"
    hash: str         # 提交哈希
    is_default: bool  # 是否为默认分支
```

### GitContent

```python
@dataclass
class GitContent:
    name: str                   # 文件/目录名
    path: str                   # 路径
    type: str                   # "file" / "dir" / "symlink" / "submodule"
    size: int                   # 文件大小（字节）
    hash: str                   # Git 哈希
    content: str                # 文件内容（仅小文件）
    entries: List[GitContent]   # 子目录内容（仅目录）
```

### DataKey

```python
@dataclass
class DataKey:
    plaintext_key: bytes   # 明文密钥
    encrypted_key: str     # 加密后的密钥（存储在文件头部）
```
