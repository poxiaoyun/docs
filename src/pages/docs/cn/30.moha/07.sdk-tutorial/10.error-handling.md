## 异常类型

Moha SDK 定义了以下异常类，继承关系如下：

```
HubException (基类)
├── RepositoryNotFoundError
├── FileNotFoundError
├── AuthenticationError
├── HTTPError
└── UploadError
```

### HubException

所有 Moha SDK 异常的基类。

**用法**：
```python
from xiaoshiai_hub import HubException

try:
    # SDK 操作
    pass
except HubException as e:
    print(f"Hub 操作失败: {e}")
```

### RepositoryNotFoundError

仓库不存在或无访问权限时触发。

**常见场景**：
- 仓库 ID 拼写错误
- 仓库已被删除
- 无权限访问私有仓库
- Hub 服务地址配置错误

**示例**：
```python
from xiaoshiai_hub import moha_hub_download, RepositoryNotFoundError

try:
    model = moha_hub_download(
        repo_id="non-existent/model"
    )
except RepositoryNotFoundError:
    print("✗ 仓库不存在")
    # 建议：检查仓库 ID 是否正确
```

### FileNotFoundError

文件不存在时触发（不同于 Python 原生的 FileNotFoundError）。

**常见场景**：
- 指定的文件在仓库中不存在
- 分支或标签不存在
- 文件已被删除

**示例**：
```python
from xiaoshiai_hub import moha_hub_download, FileNotFoundError

try:
    config = moha_hub_download(
        repo_id="ai-lab/my-model",
        filename="non-existent.json"
    )
except FileNotFoundError:
    print("✗ 文件不存在")
    # 建议：列出仓库文件查看可用文件
```

### AuthenticationError

认证失败时触发。

**常见场景**：
- Token 无效或已过期
- 用户名/密码错误
- 无认证信息时访问私有仓库
- 认证信息格式错误

**示例**：
```python
from xiaoshiai_hub import HubClient, AuthenticationError

try:
    client = HubClient(token="invalid_token")
    client.list_repositories(repo_type="models")
except AuthenticationError:
    print("✗ 认证失败")
    print("建议：检查 Token 是否有效或已过期")
```

### HTTPError

HTTP 请求失败时触发（网络错误、服务器错误等）。

**常见场景**：
- 网络连接失败
- Hub 服务不可用
- 请求超时
- 服务器返回 4xx/5xx 错误

**示例**：
```python
from xiaoshiai_hub import HubClient, HTTPError

try:
    client = HubClient(base_url="https://invalid-domain")
    client.list_repositories(repo_type="models")
except HTTPError as e:
    print(f"✗ HTTP 错误: {e.status_code}")
    if e.status_code >= 500:
        print("服务器错误，请稍后重试")
    else:
        print("请求错误，检查参数")
```

### UploadError

上传操作失败时触发。

**常见场景**：
- 上传文件大小超限
- 磁盘空间不足
- 网络中断
- 权限不足

**示例**：
```python
from xiaoshiai_hub import upload_file, UploadError

try:
    upload_file(
        path_file="huge_file.bin",
        path_in_repo="huge_file.bin",
        repo_id="my-org/repo"
    )
except UploadError as e:
    print(f"✗ 上传失败: {e}")
    # 建议：检查文件大小、网络连接
```

## 错误处理最佳实践

### 模式 1：基础错误捕获

```python
from xiaoshiai_hub import (
    moha_hub_download,
    HubException
)

try:
    model = moha_hub_download(
        repo_id="ai-lab/model",
        filename="config.json"
    )
except HubException as e:
    print(f"错误: {e}")
    # 根据错误类型处理
```

### 模式 2：详细错误处理

```python
from xiaoshiai_hub import (
    moha_hub_download,
    RepositoryNotFoundError,
    FileNotFoundError,
    AuthenticationError,
    HTTPError
)

try:
    model = moha_hub_download(
        repo_id="ai-lab/model",
        filename="config.json"
    )
except RepositoryNotFoundError:
    print("✗ 仓库不存在或无权限")
    # 处理策略：检查仓库名、权限
except FileNotFoundError:
    print("✗ 文件不存在")
    # 处理策略：列出可用文件
except AuthenticationError:
    print("✗ 认证失败")
    # 处理策略：重新登录
except HTTPError as e:
    print(f"✗ 网络错误: {e.status_code}")
    # 处理策略：重试、切换网络
```

### 模式 3：重试机制

```python
import time
from xiaoshiai_hub import moha_hub_download, HTTPError

def download_with_retry(repo_id, filename, max_retries=3):
    for attempt in range(max_retries):
        try:
            return moha_hub_download(
                repo_id=repo_id,
                filename=filename
            )
        except HTTPError as e:
            if attempt < max_retries - 1:
                wait_time = 2 ** attempt  # 指数退避
                print(f"第 {attempt + 1} 次失败，{wait_time} 秒后重试...")
                time.sleep(wait_time)
            else:
                raise

# 使用
try:
    model = download_with_retry("ai-lab/model", "config.json")
except Exception as e:
    print(f"最终失败: {e}")
```

### 模式 4：日志记录

```python
import logging
from xiaoshiai_hub import HubClient, HubException

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

client = HubClient()

try:
    models = client.list_repositories(repo_type="models")
    logger.info(f"成功获取 {len(models)} 个模型")
except HubException as e:
    logger.error(f"获取模型列表失败: {e}", exc_info=True)
    raise
```

## 常见问题 (FAQ)

### Q1: "AuthenticationError: Authentication failed" 怎么解决？

**症状**：
```
xiaoshiai_hub.exceptions.AuthenticationError: Authentication failed
```

**可能原因**：
1. Token 无效或过期
2. Token 未设置
3. 用户名/密码错误
4. Hub 地址配置错误

**解决步骤**：

```bash
# 步骤 1：检查 Token 是否设置
echo $MOHA_TOKEN

# 步骤 2：重新生成 Token
moha login

# 步骤 3：验证 Hub 地址
echo $MOHA_ENDPOINT

# 步骤 4：测试认证
python -c "from xiaoshiai_hub import load_token; print(load_token())"
```

### Q2: "RepositoryNotFoundError" 怎么解决？

**症状**：
```
xiaoshiai_hub.exceptions.RepositoryNotFoundError: Resource not found
```

**可能原因**：
1. 仓库 ID 拼写错误
2. 仓库已被删除
3. 无权限访问私有仓库
4. 仓库不存在

**解决步骤**：

```python
from xiaoshiai_hub import HubClient

client = HubClient()

# 列出所有可用模型，找到正确的仓库 ID
models = client.list_repositories(repo_type="models")
for model in models[:10]:  # 显示前 10 个
    print(f"{model.organization}/{model.name}")
```

### Q3: 下载失败如何重试？

**症状**：
```
HTTPError: Request failed
```

**解决方案**：

```python
from xiaoshiai_hub import moha_hub_download

# 通过重试机制提高成功率
model = moha_hub_download(
    repo_id="ai-lab/large-model",
    filename="model.bin"
)

# 如失败可配合重试逻辑（见下方 Q6）
```

### Q4: 上传文件失败，提示磁盘空间不足？

**症状**：
```
UploadError: No space left on device
```

**原因**：服务器磁盘空间不足

**解决方案**：

```bash
# 检查服务器磁盘
df -h

# 或分割为多个上传（上传到不同仓库或目录）
moha upload org/model ./model_part1
moha upload org/model ./model_part2
```

### Q5: 上传文件指纹校验失败？

**症状**：
```
UploadError: File integrity check failed
```

**可能原因**：
- 上传中途网络中断
- 本地文件被修改

**解决方案**：

```bash
# 重新上传（SDK 会自动续传或重新上传）
moha upload org/repo ./my_model
```

### Q6: 如何处理网络不稳定的情况？

**症状**：
频繁出现 `HTTPError` 或 `timeout`

**解决方案**：

```python
import time
import random
from xiaoshiai_hub import moha_hub_download, HTTPError

def robust_download(repo_id, filename, max_retries=5):
    """网络不稳定情况下的鲁棒下载"""
    for attempt in range(max_retries):
        try:
            return moha_hub_download(
                repo_id=repo_id,
                filename=filename
            )
        except HTTPError as e:
            if attempt < max_retries - 1:
                # 指数退避 + 随机抖动
                wait_time = min(2 ** attempt + random.random(), 60)
                print(f"重试 {attempt + 1}/{max_retries}，等待 {wait_time:.1f} 秒...")
                time.sleep(wait_time)
            else:
                raise

# 使用
model = robust_download("ai-lab/model", "config.json")
```

### Q7: 如何检查上传进度？

**症状**：不知道上传是否继续进行

**解决方案**：

```bash
# 默认会显示进度条
moha upload org/repo ./my_model

# 或查看日志
moha upload org/repo ./my_model 2>&1 | tee upload.log
```

### Q8: 如何清理本地缓存？

**症状**：本地缓存过大，占用磁盘空间

**解决方案**：

```bash
# SDK 默认 token 缓存位置
rm -rf ~/.moha/token.json

# 或直接退出登录清理本地 token
moha logout
```

### Q9: 如何使用代理访问 Hub？

**症状**：需要通过代理服务器访问

**解决方案**：

```python
import os
from xiaoshiai_hub import HubClient

# 设置代理
os.environ['HTTP_PROXY'] = 'http://proxy.example.com:8080'
os.environ['HTTPS_PROXY'] = 'https://proxy.example.com:8080'

client = HubClient()
# 后续请求会通过代理
```

### Q10: 如何启用调试日志？

**症状**：需要详细调试信息

**解决方案**：

```python
import logging

# 启用 DEBUG 日志
logging.basicConfig(level=logging.DEBUG)

# 或仅针对 SDK
logger = logging.getLogger('xiaoshiai_hub')
logger.setLevel(logging.DEBUG)

handler = logging.StreamHandler()
handler.setLevel(logging.DEBUG)
logger.addHandler(handler)

# 现在 SDK 会输出详细日志
from xiaoshiai_hub import HubClient
client = HubClient()
client.list_repositories(repo_type="models")
```

## 性能优化建议

### 1. 并发操作

```python
from concurrent.futures import ThreadPoolExecutor
from xiaoshiai_hub import moha_hub_download

files = ["file1.json", "file2.json", "file3.json"]

def download_file(filename):
    return moha_hub_download(
        repo_id="ai-lab/model",
        filename=filename
    )

# 并发下载多个文件
with ThreadPoolExecutor(max_workers=4) as executor:
    results = executor.map(download_file, files)
    for result in results:
        print(f"已下载: {result}")
```

### 2. 缓存利用

```python
from xiaoshiai_hub import moha_hub_download

# 第一次下载会缓存到本地
model = moha_hub_download("ai-lab/model", "config.json")

# 后续调用自动使用缓存，无需重复下载
model = moha_hub_download("ai-lab/model", "config.json")
```

### 3. 流式处理大文件

```python
from xiaoshiai_hub import moha_hub_download

# 下载后逐行处理，避免一次性加载到内存
filepath = moha_hub_download(
    repo_id="ai-lab/dataset",
    filename="large_data.jsonl"
)

with open(filepath) as f:
    for line in f:
        # 处理每一行
        data = json.loads(line)
```
