## 概述

创建和上传数据集到魔哈仓库。

## 创建步骤

1. **准备数据文件**：整理数据文件和目录结构
2. **填写元数据**：数据集名称、描述、标签等
3. **上传数据**：支持多种上传方式
4. **配置数据格式**：说明数据格式和使用方法

## 支持的数据格式

- CSV, JSON, Parquet
- 图像（JPEG, PNG, WebP）
- 音频（WAV, MP3, FLAC）
- 视频（MP4, AVI）
- 文本（TXT, Markdown）

## 目录结构建议

```
my-dataset/
├── train/
│   ├── images/
│   └── labels.csv
├── val/
│   ├── images/
│   └── labels.csv
├── test/
│   ├── images/
│   └── labels.csv
└── README.md
```

## 命令行上传

```bash
moha-cli dataset create --name my-dataset --path ./my-dataset
```
