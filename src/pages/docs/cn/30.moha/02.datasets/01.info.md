## 数据集概览

数据集广场展示平台上所有公开的数据集，支持搜索和分类浏览。

## 进入路径

Console → Moha → **数据集**

## 页面功能

### 数据集列表

- **搜索**：按名称和关键词搜索数据集
- **筛选**：按标签、数据类型、许可证筛选
- **排序**：按更新时间、下载量、收藏数排序

### 数据集卡片

| 信息 | 说明 |
|------|------|
| 数据集名称 | 组织/数据集名 |
| 描述 | 数据集用途简介 |
| 标签 | 数据类型、领域、语言等 |
| 文件数 | 数据集包含的文件数量 |
| 大小 | 数据集总大小 |
| 更新时间 | 最后更新时间 |

### 收藏

点击 ⭐ 按钮收藏数据集，方便后续查找使用。
