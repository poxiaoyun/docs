## 概述
审核流程确保发布至魔哈广场的模型满足安全与合规要求。本章节介绍审核人如何分配任务、核查指标并给出决策。

## 前置条件
- 模型版本处于 `Reviewing` 状态。
- 审核人具备 `ModelReviewer` 角色。

## 操作步骤
1. **领取任务**：在“审核中心”查看待审核列表，领取指定模型版本。
2. **检查内容**：核对模型介绍、数据来源、License 与安全声明。
3. **自动检测**：触发自动化脚本，扫描敏感信息与依赖漏洞。
4. **决策**：选择通过/驳回，填写意见并@ 发布者。

## 示例
```bash
rune model review \
  --model llama2-7b \
  --version 1.1.0 \
  --result approve \
  --comment "指标达标，可对外发布"
```

## 常见问题
- **Q：如何转派审核人？**
  **A：** 在审核详情中点击“转派”并选择新的负责人。
- **Q：自动检测失败？**
  **A：** 查看日志，确认检测脚本镜像可访问；必要时重试或联系平台工程师。
