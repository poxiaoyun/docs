# 模型创建

上传和注册新模型到魔哈仓库。

## 创建步骤

1. **准备模型文件**：确保模型文件格式符合要求
2. **填写基本信息**：模型名称、描述、标签等
3. **上传模型文件**：支持多种上传方式（Web、CLI、API）
4. **配置模型参数**：设置模型的输入输出格式

## 支持的模型格式

- PyTorch (.pt, .pth)
- TensorFlow (.pb, .h5)
- ONNX (.onnx)
- Safetensors (.safetensors)

## 命令行上传

```bash
moha-cli model create --name my-model --path ./model.pt
```

## API 上传

```python
from moha import Client

client = Client(api_key="your_api_key")
model = client.create_model(
    name="my-model",
    description="My awesome model",
    files=["./model.pt"]
)
```
