## 概述

GPU/CPU 资源申请和调度，优化计算资源使用。

## 资源配置

选择计算资源：

- **CPU**：2-32 核心
- **内存**：4GB-128GB
- **GPU**：T4, V100, A100
- **存储**：10GB-1TB

## 资源申请

1. 创建 Notebook 时选择资源
2. 运行时可以调整资源配置
3. 按需付费或使用配额

## 资源监控

实时查看资源使用情况：

- CPU 使用率
- 内存占用
- GPU 利用率
- 磁盘空间

## 自动关机

设置空闲自动关机，节省资源：

- 空闲超过 30 分钟自动关机
- 手动设置关机时间
- 保存工作进度

## 示例：查看 GPU 信息

```python
import torch

# 检查 GPU 可用性
print(f"GPU 可用: {torch.cuda.is_available()}")
print(f"GPU 数量: {torch.cuda.device_count()}")
print(f"当前 GPU: {torch.cuda.get_device_name(0)}")
```
