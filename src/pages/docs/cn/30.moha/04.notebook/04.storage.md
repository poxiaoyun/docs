## 概述

挂载存储卷保存工作进度，确保数据持久化。

## 存储类型

- **个人存储**：每个用户的私有存储空间
- **团队存储**：团队共享的存储空间
- **项目存储**：项目关联的存储空间

## 挂载存储卷

在 Notebook 中挂载存储：

1. 选择要挂载的存储卷
2. 指定挂载路径
3. 设置读写权限

## 数据同步

- 自动保存 Notebook
- 定期备份数据
- 同步到云存储

## 数据迁移

在不同环境间迁移数据：

- 导出数据到存储卷
- 下载到本地
- 上传到其他项目

## 示例：使用存储卷

```python
import os

# 检查存储卷挂载
storage_path = "/mnt/data"
if os.path.exists(storage_path):
    print(f"存储卷已挂载: {storage_path}")
    
    # 保存数据到存储卷
    with open(f"{storage_path}/data.txt", "w") as f:
        f.write("Hello, persistent storage!")
```

## 最佳实践

- 定期保存重要数据
- 使用版本控制管理代码
- 清理不需要的临时文件
- 备份关键数据
