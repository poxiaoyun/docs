## 应用打包

将自定义应用打包为 Helm Chart 格式，准备发布到应用市场。

## Chart 结构

```
my-app/
├── Chart.yaml          # 应用元信息
├── values.yaml         # 默认配置
├── values.schema.json  # 配置表单 Schema
├── README.md           # 应用文档
└── templates/          # Kubernetes 资源模板
    ├── deployment.yaml
    ├── service.yaml
    └── ingress.yaml
```

## 关键文件说明

### Chart.yaml

```yaml
apiVersion: v2
name: my-app
version: 1.0.0
appVersion: "2.0"
description: 我的自定义应用
```

### values.yaml

定义用户可配置的参数及默认值。

### values.schema.json

定义配置表单的 UI 呈现方式和校验规则，部署时系统会据此渲染配置表单。

## 打包命令

```bash
helm package my-app/
# 输出: my-app-1.0.0.tgz
```

:::tip
打包前建议使用 `helm lint my-app/` 检查 Chart 的格式是否正确。
:::
