## 创建应用

将自定义应用发布到应用市场。

## 前置条件

- 拥有 **系统管理员** 或 **租户管理员** 角色
- 已准备好应用的 Helm Chart 包

## 创建步骤

1. 进入应用市场 → 点击 **发布应用**
2. 填写应用基本信息：

| 字段 | 说明 |
|------|------|
| 应用名称 | 应用的唯一标识 |
| 显示名称 | 在市场中展示的名称 |
| 分类 | 选择应用所属类别 |
| 图标 | 上传应用图标 |
| 描述 | 应用功能说明 |
| 文档 | 使用说明和配置指南 |

3. 上传 Helm Chart 包
4. 配置默认部署参数
5. 点击 **发布**

:::tip
发布前建议先在测试环境中验证部署流程，确保 Chart 包和默认参数正确。
:::
