# 协作

模型、数据集、镜像和 Space 虽然资源类型不同，但在魔哈中共享统一的仓库详情框架。用户进入任一仓库后，都会围绕概览、文件、讨论、标签与设置开展协作。

## 统一标签页

根据资源类型不同，详情页会显示以下标签中的一部分：

| 标签 | 适用范围 | 说明 |
| --- | --- | --- |
| 概览 | 全部资源 | 查看仓库基础信息和 README |
| 文件 | 模型、数据集、Space | 浏览文件树、预览文件内容与提交记录 |
| 标签 | 镜像 | 查看镜像标签信息 |
| 讨论 | 模型、数据集、Space | 发起讨论、问题或 Pull Request 协作 |
| 设置 | 具备权限的成员 | 修改可见性、别名、成员、元数据和删除仓库 |
