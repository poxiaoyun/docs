# 仓库设置与发布

设置页用于维护仓库的基础属性和协作权限，不同资源类型会显示不同的配置卡片。

## 通用设置项

| 设置项 | 说明 |
| --- | --- |
| 可见性 | 设置公开、内部或私有访问范围 |
| 别名 | 更新面向用户展示的名称 |
| 成员 | 管理仓库成员及其权限 |
| 删除仓库 | 删除当前资源仓库 |

## 按资源类型差异化显示

| 资源类型 | 典型设置项 |
| --- | --- |
| 模型/数据集 | 元数据配置 |
| 镜像 | 镜像元数据、README |
| Space | 代码类元数据、封面、部署配置 |

## Space 部署

Space 是目前设置页里最特殊的资源类型，具备额外的部署卡片，可用于：

- 配置部署所需参数
- 选择联动的运行资源
- 将 Space 交付到 Rune 等运行环境

> ⚠️ 注意: 设置页往往直接影响资源可见性和团队协作权限，建议由仓库 Owner 或管理员统一维护。
