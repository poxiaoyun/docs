# 文件浏览与版本查看

文件页是仓库协作的核心页面之一，适合查看目录结构、预览文件内容以及追踪历史变更。

## 支持能力

| 能力 | 说明 |
| --- | --- |
| 文件树浏览 | 浏览目录与子文件 |
| 文件内容预览 | 根据文件类型切换目录视图或文件视图 |
| 提交记录 | 查看指定引用下的提交历史 |
| 差异对比 | 查看文件在不同版本间的差异 |
| 下载与拉取 | 结合 Token 做文件下载或 Git 操作 |

## 典型路径

- 文件树：`/file/:reference`
- 指定文件：`/file/:reference/:file`
- 提交记录：`/file/:reference/commits`
- 差异对比：`/file/:reference/diff`

## 使用建议

1. 先选定引用版本或分支。
2. 浏览目录找到目标文件。
3. 需要审查历史时进入提交记录或差异页面。
4. 如果是多人协作，配合讨论页记录变更背景。
