# 讨论与协作

讨论页用于围绕仓库内容展开问题、反馈和协作，是团队沟通的主入口。

## 页面能力

| 能力 | 说明 |
| --- | --- |
| 讨论列表 | 展示当前仓库下的讨论项 |
| 新建讨论 | 创建新的讨论主题 |
| 回复协作 | 在详情中继续回复或编辑 |
| Pull Request 入口 | 某些文件协作流会从差异页进入 PR 讨论 |

## 交互特点

- 桌面端直接显示“新建讨论”按钮。
- 移动端会将操作收纳到菜单中。
- 列表支持分页或继续加载。

## 适用场景

- 记录模型或数据集使用问题。
- 对仓库文件变更进行评审。
- 在协作过程中沉淀上下文，减少口头沟通丢失。
