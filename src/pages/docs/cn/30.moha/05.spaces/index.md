# Space 工作空间

Space 用于承载可交互的应用或演示界面，是魔哈中最接近“可运行产物”的资源类型。

## 进入路径

魔哈顶部导航 -> Space

路径：`/moha/spaces`

## 核心能力

| 能力 | 说明 |
| --- | --- |
| 列表筛选 | 支持公开/私有切换、搜索、排序和筛选 |
| 仓库详情 | 查看概览、文件、讨论和设置 |
| 部署入口 | 在设置页为 Space 配置自动部署或联动 Rune |
| 团队协作 | 通过成员、讨论和文件变更实现协作 |
