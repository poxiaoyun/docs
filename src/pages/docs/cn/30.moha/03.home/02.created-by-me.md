# 我创建的资源

“我创建的”是魔哈中的个人资产视图，用于按资源类型查看自己维护的模型、数据集、镜像和 Space。

## 相关入口

| 入口 | 说明 |
| --- | --- |
| `/moha/home/created-by-me` | 查看按类型聚合的我创建的资源 |
| `/moha/my/models` | 查看我创建的模型 |
| `/moha/my/datasets` | 查看我创建的数据集 |
| `/moha/my/images` | 查看我创建的镜像 |
| `/moha/my/spaces` | 查看我创建的 Space |

## 页面特征

- 支持按资源类型切换。
- 支持从当前页直接新建对应资源。
- 顶部标签会显示不同资源类型的统计数量。
- 右侧列表沿用统一的数据卡片样式，便于快速进入详情。

## 适用场景

- 回到自己负责维护的仓库继续更新内容。
- 快速检查某类资源是否已创建成功。
- 从个人视角统一管理不同类型资产。
