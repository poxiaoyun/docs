# 组织空间

组织空间用于从团队视角管理资源。每个组织下都可以查看模型、数据集、镜像和 Space 四类资源，并共享组织头像、描述和资源统计。

## 进入路径

魔哈首页侧栏 -> 组织主页

路径：`/moha/organizations/:organization`

## 页面结构

| 区块 | 说明 |
| --- | --- |
| 组织信息卡 | 展示组织头像、别名和描述 |
| 资源标签页 | 在模型、数据集、镜像、Space 之间切换 |
| 组织资源列表 | 展示当前组织下该类型资源 |
| 其他组织列表 | 便于在多个组织间切换 |

## 资源视图

组织页顶部标签会展示各类资源统计，适合：

- 从团队视角查看资源沉淀情况。
- 统一浏览组织公开或内部共享资源。
- 进入具体仓库继续处理文件、讨论和设置。

> 💡 提示: 如果你只想看自己创建的资源，更适合使用“我创建的”或“我的资源”视图；如果要看整个团队资产，请使用组织空间。
