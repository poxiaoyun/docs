# 首页与个人工作台

魔哈首页不是单纯的落地页，而是一套围绕“发现资源、管理个人资产、进入组织空间”的工作台。

## 页面组成

| 区域 | 说明 |
| --- | --- |
| 首页概览 | 推荐资源、横幅、快捷创建入口、公告 |
| 我创建的 | 按资源类型查看自己创建的模型、数据集、镜像和 Space |
| 我的收藏 | 聚合收藏过的资源，便于二次访问 |
| 访问令牌 | 复制当前登录态下的访问令牌 |
| 公告 | 查看平台公告详情 |
| 组织主页 | 进入某个组织下的资源列表 |
