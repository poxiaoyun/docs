# 首页概览

首页概览页聚合了资源发现与快捷操作能力，适合作为进入魔哈后的第一个页面。

## 进入路径

魔哈 -> 首页 -> 概览

路径：`/moha/home/overview`

## 页面区块

根据前端实现，首页概览主要包含以下内容：

| 区块 | 说明 |
| --- | --- |
| Banner 轮播 | 展示平台横幅，可跳转到外部链接或专题内容 |
| 快捷创建卡片 | 快速创建模型、数据集、镜像和 Space |
| 推荐资源 | 按当前分页条件展示推荐模型或资源列表 |
| 公告列表 | 展示生效中的公告，可打开详情弹窗 |

## 快捷创建

首页会提供四个直接入口：

- 新建模型
- 新建数据集
- 新建镜像
- 新建 Space

这四个入口会直接跳转到对应的创建页面，适合已有明确目标时使用。

## 适用场景

- 新用户首次进入产品，先了解平台正在重点推荐什么资源。
- 维护者需要快速新建某类资源时，从首页直接进入创建流程。
- 关注公告、活动、专题内容或资源推荐。
