# 访问令牌与公告

魔哈首页侧栏提供了两个高频辅助页面：访问令牌和公告。

## 访问令牌

路径：`/moha/home/token`

访问令牌页会直接展示当前登录态对应的 Token，并提供复制按钮。常见用途：

- Git 拉取或推送资源仓库。
- 通过脚本或 SDK 调用魔哈接口。
- 在其他平台配置访问凭据。

> ⚠️ 注意: 访问令牌属于敏感凭据，建议仅在受信环境中复制和使用。

## 公告

路径：`/moha/home/announcements`

公告页以表格形式展示当前生效的公告，支持：

- 按标题搜索
- 查看公告摘要
- 查看公告生效时间
- 打开详情弹窗阅读全文

公告适合用于发布平台更新、维护通知、活动信息或治理策略调整。
