## 概述

Kubernetes（简称 k8s）是一个开源的容器编排平台，用于自动化部署、扩展和管理容器化应用程序。

DCU Kubernetes 组件用于支持在 k8s 上使用及监控 DCU。通过这些组件，用户可以在 Kubernetes 集群中灵活分配、管理和监控 DCU 或虚拟 DCU（vDCU）资源。

## Kubernetes 组件

- **DCU-Label-Node**：以容器形式发布，并以 DaemonSet 形式部署；运行时检查节点是否存在 DCU，若存在则自动创建节点标签描述 DCU 信息。
- **DCU-Exporter**：提供 DCU/vDCU 设备监控指标并支持 Prometheus 集成，可实时监控温度、功耗、显存使用率等。
- **DCU-Device-Plugin**：基于 Kubernetes Device Plugin 框架，将 DCU/vDCU 暴露给 k8s 完成资源注册、调度与分配；支持物理卡、预切分 vDCU 与动态切分 vDCU，并兼容 HAMi。
- **vDCU-Scheduler**：由 `k8s-vdcu-admission-webhook` 与 `k8s-vdcu-scheduler` 组成；与 DCU-Device-Plugin 协同完成 vDCU 动态分配，且与 HAMi 调度器可无缝切换。

## 系统要求

| 类别 | 要求 |
| --- | --- |
| 硬件要求 | 安装海光 DCU，目前支持的 DCU 型号为：Z100L、K100/K100-AI/K100-AI-ECO、BW 系列 |
| 软件要求 | 驱动版本 ≥ 6.2.26；k8s 版本 ≥ 1.18 |
| OS 要求 | Linux（建议 Ubuntu 20.04 及以上、CentOS 7.6 及以上）；不支持 Windows |
