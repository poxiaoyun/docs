## 概述
该清单按模块归纳 Rune/BOSS/魔哈常见报错，提供诊断脚本与建议操作。

## 常见问题
1. **无法登录控制台**
   - 确认使用单点登录时浏览器未拦截三方 Cookie。
   - 若提示账号冻结，请联系租户管理员解锁。
2. **推理请求 5xx**
   - 查看推理托管的应用日志 `rune logs -f <service>`。
   - 检查网关审核是否通过及证书是否过期。
3. **配额不足**
   - 在 Rune → 设置 → 配额中查看资源占用。
   - 提交配额申请，或释放空闲工作负载。

## 示例脚本
```bash
# 检查工作负载最近 50 行日志
rune workloads logs demo-train --tail 50
```
