# Rune 智算平台

Rune 是面向模型开发、训练调优、推理部署与应用交付的一体化 AI 工作台。用户通过统一的租户、集群、工作空间上下文进入平台，在同一套控制台中完成资源准备、实例创建、运行观测和服务发布。

## 产品结构

当前前端产品主要由以下几部分组成：

| 模块 | 作用 | 典型入口 |
| --- | --- | --- |
| 开始使用 | 了解平台基础概念、准备资源和创建第一个工作负载 | `/docs/rune/guide` |
| 控制台 | 日常使用的核心工作台，覆盖仪表盘、推理、微调、开发环境、应用、实验与评测 | `/docs/rune/console` |
| 对话应用 | 提供模型体验、多模型对比、参数调试与 Token 估算 | `/docs/rune/chatapp` |
| 资源管理 | 查看模板、配额与平台侧资源策略 | `/docs/rune/resources` |

## 典型用户路径

1. 由管理员为租户分配集群、配额和工作空间。
2. 用户进入 Rune 控制台，先选择集群和工作空间上下文。
3. 在应用市场或模板中心选择模板，创建推理、微调、开发环境或应用实例。
4. 在实例详情中查看状态、监控、日志、事件和访问地址。
5. 结合实验、评测和 ChatApp 完成模型验证，再将服务交付给团队使用。
