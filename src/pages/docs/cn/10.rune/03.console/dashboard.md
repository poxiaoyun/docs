# 仪表盘

Rune 仪表盘是进入工作台后的默认首页，用于展示当前租户、集群和工作空间上下文下的资源概览与常用入口。

## 进入路径

Rune 工作台 -> 仪表盘

路径：`/rune/dashboard`

## 使用前提

- 已登录并选择租户。
- 当前账号具备 Rune 产品访问权限。
- 如平台启用了区域和工作空间隔离，需在顶部先选择集群与工作空间。

## 页面重点

仪表盘通常承担三类信息展示：

| 区块 | 说明 |
| --- | --- |
| 概览卡片 | 汇总当前上下文中的实例数量、资源使用或服务状态 |
| 最近活动 | 展示最近创建、更新或运行中的实例活动 |
| 资源使用趋势 | 直观查看 CPU、内存、GPU 等资源消耗情况 |

## 顶部上下文选择

仪表盘左上角会结合产品配置展示：

- 集群选择器：切换当前可操作的集群。
- 工作空间选择器：切换当前工作空间，所有实例列表随之变化。

> 💡 提示: 如果列表页显示为空，优先检查是否切到了错误的集群或工作空间。

## 常见操作

1. 进入仪表盘确认当前工作上下文。
2. 从仪表盘卡片判断资源是否充足。
3. 通过最近活动快速跳转到推理、微调、开发环境或应用详情页。
4. 如果发现实例异常，再进入日志或实例详情排查。

## 适用场景

- 团队日常巡检当前工作空间的资源状态。
- 快速定位最近创建或运行中的实例。
- 作为进入推理、微调、实验与评测模块的起点。
