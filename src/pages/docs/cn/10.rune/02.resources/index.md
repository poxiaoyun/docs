# 资源管理

`资源管理` 栏目对应 Rune 中偏“共享资源与治理”的能力，重点不是单个实例的运行过程，而是实例创建前后会反复复用的模板、配额和规格。

## 与控制台的区别

| 栏目 | 关注重点 |
| --- | --- |
| 控制台 | 推理、微调、开发环境、应用、日志、存储等日常操作 |
| 资源管理 | 实例模板复用、租户与工作空间配额、可用算力规格 |

## 当前覆盖范围

| 页面 | 说明 |
| --- | --- |
| [实例模板与复用](/rune/resources/templates) | 将已有实例沉淀成模板，复用到新的部署任务 |
| [配额与策略](/rune/resources/quotas) | 查看租户配额，并理解工作空间配额如何逐级分配 |
| [算力规格](/rune/resources/flavors) | 查看当前租户在指定集群中可使用的资源规格 |

## 推荐阅读顺序

1. 先看模板，理解实例如何沉淀成可复用配置。
2. 再看配额，确认当前租户和工作空间的资源边界。
3. 最后看规格，选择当前集群里真正可用的资源组合。
