# 监控、日志与调度

集群详情页提供了多种运维入口，用于从不同维度观察集群状态并调整调度策略。

## 页面能力

| 页面 | 当前能力 |
| --- | --- |
| 指标监控 | 预留页面，当前前端为建设中状态 |
| 事件 | 预留页面，当前前端为建设中状态 |
| 日志 | 提供日志查询、标签过滤和实时流式查看 |
| 调度器 | 提供 Volcano 调度配置的拖拽式编排与保存 |

## 日志页

日志页使用统一日志查看器组件，支持：

- 自定义查询语句
- 标签查询与标签值联想
- 历史日志查询
- WebSocket 实时日志流

适合用于排查：

- 平台组件异常
- 集群服务启动失败
- 调度、存储或网关侧报错

## 调度器页

调度器页面向高级管理员，用于维护集群调度配置。当前前端支持：

- 加载调度参数与现有配置
- 通过拖拽调整 action 和 plugin 顺序
- 编辑本地配置副本
- 保存并下发到后端

> ⚠️ 注意: 调度配置会直接影响集群的作业调度行为，建议先在测试环境验证，再变更生产集群。
