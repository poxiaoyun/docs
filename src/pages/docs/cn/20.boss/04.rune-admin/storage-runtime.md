# 存储与运行时服务

在单集群视角下，Boss 不只管理业务工作负载，也管理平台运行所需的存储和系统级实例。

## 相关页面

| 页面 | 作用 |
| --- | --- |
| 存储实例 | 管理集群中的系统存储类实例，可新增、查看和删除 |
| 内置系统镜像 | 查看系统级实例与其运行状态 |
| 系统镜像市场 | 从市场部署新的系统实例 |

## 存储实例页

存储页以实例列表形式展示系统中的存储类组件，并提供新增入口。常见使用方式：

1. 进入集群存储页查看已有存储实例。
2. 点击新增，从系统市场选择存储类模板。
3. 查看实例详情，确认运行状态和关联资源。

## 系统市场

系统市场用于部署集群公共能力，例如监控、日志、开发支撑或其他平台级组件。它与普通租户工作负载不同，面向的是整集群或平台侧基础设施。

## 适用场景

- 初始化新集群的基础能力。
- 补装公共运行时组件。
- 排查平台级存储实例运行异常。
