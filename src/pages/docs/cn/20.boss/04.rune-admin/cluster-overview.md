# 集群总览

集群总览页是平台管理员进入单个 Rune 集群后的主视图，用于查看基础资源面板和监控仪表盘。

## 进入路径

Boss -> Rune 智算管理 -> 集群管理 -> 选择集群 -> 总览

## 页面结构

总览页基于后端下发的集群仪表盘配置渲染，默认展示基础类仪表盘，不包含单独的 GPU 仪表盘。

| 区块 | 说明 |
| --- | --- |
| 集群基础监控 | CPU、内存、节点、核心服务等基础指标 |
| 动态变量筛选 | 支持按监控模板提供的变量切换视角 |
| 下钻入口 | 从总览进入节点、GPU、存储、日志和资源页面 |

## 适用场景

- 确认集群整体运行是否健康。
- 发现资源异常后，继续进入节点或日志页面定位问题。
- 在执行扩容、升级或调度调整前，先观察当前集群负载。
