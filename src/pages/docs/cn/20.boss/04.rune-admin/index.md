## 概述

Rune 智算管理是 Boss 运营平台中用于维护 Rune 智算平台基础设施的模块。管理员可以在此管理 GPU/CPU 集群、分配资源池与配额、维护算力规格定义以及管理系统镜像。

## 章节导航

| 模块 | 说明 |
|------|------|
| [集群管理](/docs/boss/rune-admin/clusters) | 接入并管理 K8s 计算集群 |
| [资源池管理](/docs/boss/rune-admin/resource-pools) | 在集群中划分资源池，关联节点 |
| [租户与配额](/docs/boss/rune-admin/tenants) | 为租户分配资源池及 GPU/CPU 配额 |
| [算力规格管理](/docs/boss/rune-admin/flavors) | 定义和维护可选的算力规格模板 |
| [私有镜像](/docs/boss/rune-admin/templates) | 租户自定义镜像管理 |
| [系统镜像市场](/docs/boss/rune-admin/system-market) | 公开的系统镜像市场管理 |
| [内置系统镜像](/docs/boss/rune-admin/systems) | 平台预置的基础镜像维护 |
