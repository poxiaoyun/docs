## 概述
租户管理是平台多租隔离的基础。BOSS 提供租户生命周期管理、配额发放与审计能力，支持与 LDAP/SSO 集成。

## 前置条件
- 已创建至少一个集群。
- 管理员具备 `PlatformAdmin` 角色。

## 操作步骤
1. **创建租户**：输入名称、计费主体与联系人，选择默认集群与可用区域。
2. **绑定项目**：在租户详情中新建项目空间，并分配管理员。
3. **发放配额**：设置 GPU/CPU/存储等上限，可按项目拆分。
4. **审计**：启用操作日志与成本对账，确保行为可追溯。

## 示例
```bash
rune tenant create \
  --name ai-research \
  --cluster gpu-prod \
  --quota "gpu=200,cpu=500"
```

## 常见问题
- **Q：租户创建失败提示名称占用？**
  **A：** 名称需全局唯一，可使用前缀标识业务线。
- **Q：如何批量导入成员？**
  **A：** 通过“同步目录”功能绑定企业 LDAP，或上传 CSV。
