# 许可证管理

许可证页面用于查看当前产品授权状态，并协助管理员在授权即将到期或已经失效时进行处理。

## 进入路径

Boss -> 系统设置 -> 许可证

路径：`/settings/license`

## 页面作用

| 能力 | 说明 |
| --- | --- |
| 授权状态查看 | 查看许可证是否正常、生效时间和到期时间 |
| 到期提醒 | 当许可证即将到期或已过期时，在产品内给出告警 |
| 续期协同 | 提醒管理员联系平台供应方更新授权文件或密钥 |

## 常见状态

| 状态 | 含义 |
| --- | --- |
| 正常 | 当前许可证可用，产品功能按授权范围开放 |
| 即将到期 | 许可证接近到期，建议提前安排续期 |
| 当日到期 | 当天到期，部分功能可能随配置受限 |
| 已过期 | 授权失效，部分产品能力可能不可用 |

## 建议处理流程

1. 在页面确认到期时间与授权范围。
2. 若收到到期提醒，尽快联系授权维护方准备新许可证。
3. 在维护窗口完成更新，并重新确认产品状态恢复正常。

> ⚠️ 注意: 许可证过期后，Boss、Rune、Moha 或 ChatApp 的部分能力可能受限，建议至少提前一周完成续期准备。
