## 颜色和视觉层次

### Hero 区块
- **背景**：浅色渐变 (主色 2% → 8%)
- **径向装饰**：两个半透明的彩色圆形光晕
- **标题**：大号加粗，渐变色文字
- **卡片**：白色底，顶部彩条，悬停上升

### 产品卡片色彩标识
```
魔哈仓库: ████ #00A76F (绿)
Rune:    ████ #1877F2 (蓝)
BOSS:    ████ #7635DC (紫)
```

### 区块背景交替
- 白色区块：Hero、能力总览、解决方案、生态资源、社区
- 灰色区块：快速上手 (alpha 4%)
- 深色区块：CTA (主色渐变)

## 交互动画

### 进入动画（滚动触发）
1. 标题区：从上淡入
2. 描述文字：从上淡入（延迟）
3. 按钮/卡片：从下淡入（依次）

### 悬停效果
- **产品卡片**：
  - 上升 8px
  - 阴影从 z8 → z20
  - 顶部彩条 scaleX 0 → 1
  
- **能力卡片**：
  - 上升 4px
  - 阴影从 z4 → z16
  
- **社区卡片**：
  - 右移 4px
  - 阴影从 z4 → z12

- **按钮**：
  - 渐变背景变化
  - 轻微上升

## 响应式断点

### 移动端 (xs)
- 单列布局
- 字号缩小
- 按钮垂直堆叠
- 内边距减小

### 桌面端 (md+)
- 三列网格
- 大字号
- 按钮横向排列
- 充足留白

## 视觉对比度

### 文字层次
- **h1**: 3.5rem, 800 weight, 渐变色
- **h2**: 3rem, 800 weight
- **h3**: 2.125rem, 700 weight
- **h5**: 1.5rem, 600 weight
- **body1**: 1rem, 400 weight
- **overline**: 0.75rem, uppercase

### 阴影系统
- **z4**: 卡片默认
- **z8**: 按钮默认
- **z12**: 悬停浅
- **z16**: 悬停中
- **z20**: 悬停深

## 动画时序

```
页面加载后:
  → Hero 标题淡入 (0-300ms)
  → Hero 副标题淡入 (100-400ms)
  → Hero 按钮淡入 (200-500ms)
  → Hero 卡片依次淡入 (300-900ms)

滚动到能力总览:
  → 区块标题淡入 (0-300ms)
  → 三个卡片依次淡入 (0-600ms)

... 以此类推
```

## 品牌一致性

所有产品相关元素都使用统一的品牌色：
- 图标背景
- 边框/装饰条
- 标签颜色
- 按钮渐变
- 悬停效果

这确保了视觉上的连贯性和专业度。
